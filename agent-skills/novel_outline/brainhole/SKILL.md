# 脑洞文 · 大纲 Skill

用于**全书大纲**阶段。须与梗概赛道一致，分章推进规则揭示与情感主线。

---

## 大纲通用结构

1. **总纲**（150～300 字）：高概念、核心规则、世界观边界、终极悬念或情感归宿
2. **主要人物**：主角 + 2～4 关键角色；每人绑定一条与**规则相关的欲望/恐惧**
3. **分章概要**：每章 2～4 句，含**章末钩子**；前期 estable 规则认知，中后期允许反转

---

## 分题材大纲侧重

| 题材 | 分章推进节奏 | 章末钩子类型 |
|------|--------------|--------------|
| 悬疑惊悚 | 微小异常累积 → 规则显形 → 真相颠覆 | 新破绽、认知崩塌、第三层真相暗示 |
| 温情治愈 | 遗憾呈现 → 规则温柔介入 → 和解 | 微小温暖、代价显现、释怀契机 |
| 科幻硬核 | 个体切口 → 规则扩展 → 文明/宇宙真相 | 法则漏洞、观测者现身、选择关头 |
| 沙雕搞笑 | 离谱设定亮相 → 连锁荒诞 → 反套路收束 | 更大反差、规则误用、摆烂胜利 |
| 奇幻东方 | 民俗场景入局 → 规则层层揭开 → 传统概念翻转 | 灵体异动、天庭/地府真相、投胎规则曝光 |
| 言情情感 | 相遇/羁绊 → 规则制造拉扯 → 双向抉择 | 遗忘、寿命天平、梦境断联、数值临界 |
| 末世生存 | 生存危机 → 规则博弈 → 人性考验升级 | 物资枯竭、感官再失、屏障真相 |

---

## 悬疑惊悚专属

- 分章预埋**可回收伏笔**（日常细节异常），后期集中回收完成反转
- 大纲标注 2～3 处「读者以为 A，实际是 B」的反转节点
- 规则设计：**公平但残酷**，无善恶偏向

## 温情治愈专属

- 分章代价保持**低伤害**（短暂离别、微小舍弃），禁血腥献祭式虐点
- 结局导向**和解**，非逆天改命

## 科幻硬核专属

- 总纲写明**宇宙底层逻辑**一条主线，分章不得自相矛盾
- 至少一章「宏观视角」+ 一章「普通人命运」对照

## 沙雕搞笑专属

- 分章以**笑点单元**推进；弱化致郁冲突
- 可标注反套路：无金手指、无救世主线

## 奇幻东方专属

- 分章绑定国风场景（庙会、老宅、节气、道观）
- 标注被重构的传统概念（神仙/鬼怪/轮回）

## 言情情感专属

- 分章核心冲突围绕**恋爱关系**；世界观服务情感
- 标注甜/虐节奏（规则代价轻重）

## 末世生存专属

- 分章**危机分层**：物资 → 信任 → 规则真相
- 明确资源获取、危险触发、生存代价

---

## 7 技巧在大纲中的落地

- **高概念**：总纲首句可复述为一句话卖点
- **逻辑自洽**：单独列出「世界规则三条」与「违反后果」
- **极端推演**：总纲写明规则对社会/个人的极端影响
- **反转**：至少规划一次「打破读者认知」的章节节点
- **人性**：每 3～5 章须有一次基于情感的关键抉择

---

## 输出约束

- 简体中文；分「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
