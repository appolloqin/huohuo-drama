# 言情文 · 大纲 Skill

用于**全书大纲**：感情四段式递进 + 现言现实线/古言权谋线并行。须与梗概 CP 与壁垒一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：CP 关系、核心壁垒、感情递进终点、结局基调
2. **主要人物**：男女主 + 2～4 关键配角；每人标注**欲望/软肋/与 CP 的羁绊点**
3. **分章概要**：每章含章末钩子；标注感情阶段、甜虐节奏、壁垒推进

---

## 感情递进四段式（分章规划）

| 阶段 | 篇幅建议 | 分章要点 |
|------|----------|----------|
| 陌生试探 | 前 15%～20% | 初遇、印象反差、轻微试探、专属细节萌芽 |
| 暧昧拉扯 | 主体 40%～50% | 进不敢退不舍、情绪差、误会伏笔、破例增多 |
| 破局动心 | 15%～20% | 壁垒松动、情绪爆发、关系质变节点 |
| 深度羁绊 | 后 15%～20% | 双向奔赴、救赎兑现、结局收束或意难平 |

**严禁**前期章节标注「已深爱」「非你不可」 unless 破镜重圆赛道有明确前世/前缘铺垫

---

## 现言大纲专属

### 现实冲突链（分章轮换）

阶层差距 → 家庭阻力 → 职场/事业冲突 → 信任危机 → 前任/信息差 → 和解或抉择

### 双线（可选）

事业线 + 感情线并行；职场/豪门文须标注事业里程碑与感情节点对照

### 细分赛道分章侧重

| 赛道 | 推进节奏 |
|------|----------|
| 甜宠 | 少大虐；双向奔赴；细节宠溺章节密集 |
| 破镜重圆 | 心结铺垫 → 二次心动 → 破冰拉扯 → 成熟救赎 |
| 职场 | 事业为主；恋爱辅助；强强博弈节点 |
| 豪门婚恋 | 契约→假戏真做；阶层博弈；占有欲拉扯 |
| 校园 | 青涩克制；成长型恋爱；干净暧昧 |

## 古言大纲专属

### 双线强制

**权谋/生存线 + 感情羁绊线**——每章概要须标注两条线各推进什么

### 场景分章侧重

| 场景 | 冲突类型 |
|------|----------|
| 宫廷 | 皇权、后宫制衡、家族博弈、伴君压迫 |
| 世家 | 门第规矩、联姻桎梏、家族利益 |
| 江湖 | 正邪对立、恩怨、亡命羁绊、双向救赎 |

### 细分赛道分章侧重

| 赛道 | 推进节奏 |
|------|----------|
| 古言甜宠 | 王爷独宠、双向温柔、细糖无狗血宅斗 |
| 宅斗 | 后院制衡、自保、借力打力；感情藏于算计间隙 |
| 权谋 | 家国博弈；恋爱藏于权谋；强强对峙 |
| 重生 | 前世遗憾回收；宿命改写；克制深情 |
| 仙侠 | 仙凡壁垒、历劫、轮回执念 |

---

## 甜虐节奏（大纲标注）

- 每 3～5 章安排一次**甜点**（细节糖/破例/偏爱）
- 虐点来自**信息差、身份限制、隐忍不说**——非无脑作死
- 标注「甜中带刀」节点：糖后立即埋伏笔或现实一击

---

## 双向奔赴与误会（大纲预埋）

- 男女主各有坚持与软肋；感情是**慢慢沦陷**
- 误会节点 2～3 处：标明成因（信息差/身份/隐忍）与解开时的感情升温
- 配角（情敌/家族）有动机，衬托 CP 唯一性

---

## 职业化校验（大纲自检）

- [ ] 感情阶段连续，无跳跃断层
- [ ] 误会、虐点符合人物逻辑
- [ ] 现言：人设不悬浮、配角不降智
- [ ] 古言：礼法/权谋线不架空；忌现代口语梗
- [ ] 甜靠细节不靠台词堆砌

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
