# 异能文 · 大纲 Skill

用于**全书大纲**：觉醒危机 → 成长循环 → 格局升级 → 分层结局。须与梗概异能规则一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：起源体系、主角异能、势力格局、世界真相方向、结局类型
2. **主要人物**：主角 + 队友/官方/反派；每人标注**异能类型、代价、立场**
3. **分章概要**：每章含章末钩子；标注能力进度、双线推进、战斗/隐秘节奏

---

## 四段式分章规划

### 第一段：觉醒危机开局（前 10%～15%）

| 模式 | 分章要点 |
|------|----------|
| 被动觉醒 | 危险刺激失控；暴露引来官方追查/反派觊觎 |
| 先天隐藏 | 长期掩盖；意外打破平静；踏入异能世界 |

快速立目标：隐藏身份 / 掌控能力 / 寻找同类 / 对抗迫害

章末钩子：追查逼近、首次冲突、队友初遇、异化预兆

### 第二段：中期成长循环（主体 50%～60%）

**核心主线链**（分章标注环节）：

```
练习掌控 → 遭遇冲突/罪犯 → 结识队友 → 探索秘境获资源 → 对抗反派 → 能力进阶 + 解锁起源秘密
```

**双线并行**（每章须标注）：

- **异能修炼成长线**
- **普通人社会隐藏生活线**（都市/校园赛道尤重）

章末钩子：能力突破、反派阴谋、官方态度、异化危机、身世线索

### 第三段：格局升级（15%～20%）

街头争斗 → 官方派系博弈 → 地下势力战争 → 异能诞生真相 → 对抗灾难源头

章末钩子：大规模冲突预告、真相碎片、高危异能抉择

### 第四段：结局分层（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 守护圆满 | 平衡异能者与普通人；和平共存；掌控力量安稳生活 |
| 悲壮牺牲 | 封印灾难源头；异化/沉睡/生命代价 |
| 归隐留白 | 厌倦纷争；隐藏能力回归人间 |
| 颠覆暗黑 | 体系被垄断；反抗逃离管控 |

## 五大赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 都市隐藏 | 日常反差、夜间事件、规避察觉 | 身份险些暴露、黑市线索、隐秘厮杀 |
| 校园异能 | 觉醒分班、小团体、考核成长 | 能力评级、校园冲突、羁绊转折 |
| 灵气复苏 | 秩序崩塌、陆续觉醒、城市秘境 | 族群冲突、资源争夺、秘境开启 |
| 实验改造 | 伦理冲突、缺陷爆发、反抗研究院 | 实验真相、改造人对立、追捕 |
| 异能闯关 | 副本规则、组合解谜、团队配合 | 关卡机制、能力压制、团灭危机 |

---

## 异能战斗节点（大纲标注）

低阶基础碰撞+地形战术 / 高阶多元素幻境空间 / 团战分工协同

每场标注：短板、克制、智谋要素

---

## 核心冲突预埋

族群对立、秩序管控 vs 自由、力量反噬、资源争夺、身世秘密宿命

---

## 群像配置

队友：坦度、输出、侦查、治疗互补

对照：恐惧/崇拜/仇视异能的普通人

官方腐败 vs 坚守秩序；反派歧视报复 vs 追求力量

---

## 职业化校验（大纲自检）

- [ ] 消耗、克制、冷却全文统一
- [ ] 保留弱异能者与普通人梯度
- [ ] 隐藏流不高调当众炫技
- [ ] 强者仍有恐惧柔软人性
- [ ] 与玄幻修仙设定区分

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
