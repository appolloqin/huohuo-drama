# 谍战文 · 大纲 Skill

用于**全书大纲**：高压开局 → 博弈循环 → 破局反转 → 分层结局。须与梗概三方格局与谍战规则一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：时代背景、三方格局、主角潜伏身份、核心任务线、结局类型
2. **主要人物**：主角 + 敌我关键角色；每人标注**立场、可疑点、隐瞒动机**
3. **分章概要**：每章含章末钩子；标注任务进度、暴露风险、情报真假、反转层级

---

## 四段式分章规划

### 第一段：高压开局（前 10%～15%）

开篇即危机（标注采用哪种）：

情报断联、交通站暴露、叛徒出现、全城排查、任务紧急、身边试探

建立基调：步步为营、如履薄冰、无安全感

章末钩子：第二个试探者、假情报苗头、家人牵连预兆

### 第二段：中段博弈循环（主体 55%～65%）

**核心节奏链**（分章标注环节）：

```
接收秘密任务 → 伪装执行 → 遭遇试探排查 → 叛徒钓鱼 → 真假情报博弈 → 牺牲取舍 → 惊险过关 → 埋下更大危机
```

**原则**：每次成功伴随一次暴露风险

章末钩子：身份疑似、内鬼线索、任务代价、钓鱼圈套

### 第三段：高潮破局（15%～20%）

**三重反转堆叠**（分章标注层级）：

| 层级 | 内容 |
|------|------|
| 表层危机 | 任务失败、身份疑似暴露 |
| 中层反转 | 队友是内鬼 / 敌人早已怀疑 |
| 深层终极 | 主角早已预判、反向布局、将计就计 |

章末钩子：反杀内鬼、情报翻盘、最终暴露抉择

### 第四段：结局三类梯度（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 隐忍圆满 | 任务完成、全身而退、继续潜伏无人知晓 |
| 悲壮殉道 | 身份暴露、牺牲自己保全组织、无名英雄 |
| 意难平留白 | 胜利到来故人不在；盛世再无当年人 |

## 信任与情报（大纲标注）

- 分章标注情报真伪、传递节点、销毁环节
- 内鬼/可疑人物伏笔与回收章节
- 牺牲取舍节点（任务 vs 战友 vs 爱人）

---

## 核心冲突预埋

信任（人人可疑）、信息差（真假混杂）、心理博弈（审讯钓鱼）、取舍、身份危机

---

## 节奏与氛围节点

紧：试探、排查、审讯、突围

松：短暂伪装日常、情感压抑铺垫（极少放松）

潜台词对话章节分布；0.1秒微破绽预埋

---

## 职业化校验（大纲自检）

- [ ] 单线联系、情报流程、暴露代价全文统一
- [ ] 卧底低调平庸，非高调张扬
- [ ] 敌方高智商多疑，非降智
- [ ] 情爱不误任务逻辑
- [ ] 脱险靠布局预判牺牲，非运气
- [ ] 局中局反转有铺垫

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
