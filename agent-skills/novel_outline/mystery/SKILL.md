# 悬疑文 · 大纲 Skill

用于**全书大纲**：开局钩子 → 迷雾叠加 → 真相拆解 → 双层反转。须预埋可回收伏笔清单。

---

## 大纲通用结构

1. **总纲**（150～300 字）：案件核心、关系网、表层/深层真相方向、赛道基调
2. **主要人物**：主角 + 嫌疑人/证人/凶手；每人标注**动机、隐瞒点、误导作用**
3. **分章概要**：每章含章末钩子；标注线索层级、迷雾推进、伏笔埋设/回收节点
4. **伏笔清单**（建议附在总纲后）：道具/台词/动作/环境——对应回收章节

---

## 四段式分章规划

### 第一段：开局钩子（前 5%～10%）

| 开篇 | 分章要点 |
|------|----------|
| 尸体开局 | 直接案发；现场诡异疑点丛生 |
| 异常开局 | 日常无法解释的违和细节 |
| 预言开局 | 死亡预告、倒计时、诡异留言 |

快速建立：现场、核心疑点、主角介入契机

章末钩子：第二疑点、第一个误导线索、关系人登场

### 第二段：迷雾叠加（主体 60%～70%）

**核心循环**（分章标注环节）：

```
线索浮现 → 初步锁定嫌疑人 → 证据推翻 → 第二重迷雾 → 新死者/新线索 → 关系网崩塌 → 接近真相再反转
```

**迷雾三层**（分章标注层级）：

- 浅层疑点（读者可见）
- 中层疑点（角色困惑）
- 深层疑点（结局揭晓）

误导轮换：视角盲区、时间差、物证假象、口供真假混杂

章末钩子：推翻、新死者、关系崩塌、假突破

### 第三段：高潮真相拆解（15%～20%）

集中回收伏笔：物证吻合、时间线闭合、动机落地、凶手行为闭环

读者恍然大悟且觉得早有预兆

章末钩子：表层破案完成、深层异常萌芽

### 第四段：双层反转结局（后 5%～10%）

| 层级 | 内容 |
|------|------|
| 表层真相 | 案件告破、凶手落网 |
| 深层反转 | 真凶另有其人、幕后推手、全员误导、主角陷局 |

---

## 分赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 刑侦悬疑 | 指纹弹道尸检监控、排除法时间线 | 物证矛盾、不在场证明破裂 |
| 社会派 | 人性原生家庭、舆论；动机同情 | 受害者阴暗面、社会反思点 |
| 心理悬疑 | 记忆人格视角欺骗 | 自我认知崩塌、第三层「我」 |
| 校园悬疑 | 冷暴力流言失踪 | 青春恶意曝光、青涩阴冷转折 |
| 都市微恐 | 日常异常邻里错位 | 认知颠覆、细思极恐回收 |
| 反转悬疑 | 密集误导与回收 | 表层破案后立即深层裂缝 |

---

## 节奏松紧（大纲标注）

紧：案发、追凶、危机、对峙

松：人物过往、情感铺垫、线索复盘

每 3～5 章一张一弛

---

## 核心冲突预埋

逻辑矛盾（证据时间线动机）、人性灰度（好人阴暗坏人有苦衷）、自我怀疑（创伤幻觉记忆）、幕后连环局

---

## 职业化校验（大纲自检）

- [ ] 每条伏笔有对应回收章节
- [ ] 反转均有前文铺垫
- [ ] 凶手动机行为闭环完整
- [ ] 警察配角推理合理非降智
- [ ] 刑侦物证流程无硬伤
- [ ] 无机械降神新人物新线索终局救场

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」；建议附「伏笔清单」
- 格式：`第N章：标题 / 概要`；不要前言套话
