# 穿越文 · 大纲 Skill

用于**全书大纲**：落地危机 → 降维发育循环 → 剧情偏差/高层博弈 → 终局升华。须与梗概穿越规则一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：穿越机制、落点世界、认知优势、核心冲突、终局类型
2. **主要人物**：主角 + 原主关系网 + 原著/时代关键角色；每人标注**与穿越者的信息差关系**
3. **分章概要**：每章含章末钩子；标注发育进度、打脸节点、剧情偏差度（穿书）

---

## 四段式分章规划

### 第一段：开局落地危机（前 10%～15%）

- **黄金开局**：穿来即困境——重伤、被休、被陷害、死局、危机临门
- 拒绝冗长迷茫；立刻立目标：活下去、站稳脚、改命运
- 原主烂摊子亮相；现代灵魂水土不服初显
- 章末钩子：首个死局倒计时、关键人物登场、暴露风险萌芽

### 第二段：降维发育循环（主体 50%～60%）

**核心爽点链**（分章标注环节）：

```
利用信息差/知识差 → 解决当下危机 → 打脸偏见者 → 积累资本人脉 → 暴露异常 → 关注/猜忌 → 更高层级博弈
```

- 每次循环有升级：更大危机、更高层级、更深猜忌
- 初期须**藏拙**；出格行为标注代价
- 章末钩子：新商机/死局、打脸对象、天道/剧情修正预兆

### 第三段：剧情偏差与高层博弈（15%～25%）

**穿书专属**（若适用，分章标注偏差链）：

改小命运 → 牵动主线人物 → 原著偏移 → 未知危机 → 跳出剧本原创线

**通用**：更高层级博弈、身份暴露恐惧、三观激烈碰撞、孤独感虐点

章末钩子：剧情崩坏点、天道反噬、重大改写宿命节点

### 第四段：终局升华（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 凡人圆满 | 立足时代、富贵安稳、守护家人 |
| 格局逆袭 | 改革制度、富民强国、青史留名 |
| 时空闭环 | 回归原世界、双向遗憾、留守新世界、时空遗忘 |

## 分赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 今穿古 | 知识降维、制度冲突、低调发育、实业/权谋 | 阶级碾压危机、技术瓶颈、权贵关注 |
| 古穿今 | 三观冲击、学习现代、反差萌、烟火沦陷 | 文化冲突、身份适应、情感转折 |
| 穿书穿剧 | 预知卡点、改命、剧情偏移、对抗剧情修正力 | 原著死局变异、男主女主命运惯性 |
| 反向穿越 | 权谋吊打职场商界、理智降维 | 现代规则不适、情感线萌芽 |
| 跨界穿越 | 规则摸索、逻辑破局、无金手指求生 | 新世界规则惩罚、信息差失效 |

---

## 核心冲突预埋

三观冲突（平等 vs 尊卑陋习）

天道/剧情修正（穿书）

身份暴露恐惧（妖孽、识破、杀身之祸）

孤独感（异乡人、无人懂过往）

---

## 爽点节点分布（大纲标注）

认知差 / 翻盘 / 思维降维 / 命运逆袭 / 偏爱羁绊——每 3～5 章至少一类

---

## 职业化校验（大纲自检）

- [ ] 穿越规则代价全文统一
- [ ] 主角保留现代人底线，有成长过渡非瞬间古人化
- [ ] 穿书有剧情崩坏原创线，非全程照搬原著
- [ ] 初期藏拙，出格有代价
- [ ] 历史穿越制度重大事件无硬伤

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
