# 科幻文 · 大纲 Skill

用于**全书大纲**：开篇切入 → 微观个人线 + 宏观文明线双线推进 → 分层结局。须与梗概设定闭环一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：核心科技/宇宙规则、文明矛盾、终极真相方向、结局类型
2. **主要人物**：主角职业身份 + 2～4 关键角色；每人标注**科技相关欲望/恐惧/内在冲突**
3. **分章概要**：每章含章末钩子；标注个人线/文明线进度及设定释放层级

---

## 开篇切入（分章规划）

两种模式灵活运用（大纲标注采用哪种）：

| 模式 | 分章要点 |
|------|----------|
| 具象危机开局 | 飞船故障、外星入侵、实验失控、时空裂缝——前 1～3 章急切入 |
| 日常反差开局 | 未来都市普通生活 → 逐步暴露底层科技隐患 |

章末钩子：更大危机露头 / 规则漏洞初现 / 文明层面异动暗示

---

## 中段双线推进（主体篇幅）

**微观个人线 + 宏观文明线并行**

剧情递进链条（分章标注所处环节）：

```
技术探索 → 遭遇危机 → 发现体系漏洞 → 文明矛盾爆发 → 触碰世界终极真相
```

### 设定分层释放（长篇）

- 前期：角色可操作层面的科技规则
- 中期：文明历史、种族格局、资源战争逻辑
- 后期：宇宙真相、高层秘密、维度/高维存在

**禁止**前期大段设定说明文；通过事件与抉择自然解锁。

### 短篇大纲

聚焦单一科幻概念深挖；删减无关宏大背景，保证主题集中。

---

## 结局分层设计（最后 5%～15%）

| 结局类型 | 大纲落点 |
|----------|----------|
| 思辨留白 | 抛出终极问题，不给出标准答案 |
| 抗争史诗 | 人类突破桎梏、对抗更高维度 |
| 循环宿命 | 时空闭环、文明轮回、历史重演 |
| 温情和解 | 人与 AI、不同种族共存 |

须与开篇设定、技术代价伏笔形成闭环或有意留白。

---

## 细分赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 硬科幻 | 科研推演、星际探索、天体现象 | 实验异常、航线危机、文明存续抉择 |
| 软科幻 | 科技对社会/人性的改变 | 伦理两难、制度崩坏、关系撕裂 |
| 赛博朋克 | 义体、垄断企业、虚实割裂、底层流民 | 企业追杀、意识劫持、现实崩塌 |
| 时空穿梭 | 悖论、因果闭环、平行分割 | 历史篡改后果、自我相遇、闭环暗示 |
| 星际殖民 | 改造、开采、贸易、战争 | 殖民地叛离、环境灾难、外星接触 |
| AI 与人机 | 进化规则、意识判定、算法管控 | 觉醒事件、人权争议、人机情感临界 |

---

## 核心冲突预埋（大纲标注）

重要章节标注冲突类型（至少一类）：

- **科技伦理**：基因筛选、意识永生、人造生命、意识抹杀
- **文明层级**：低等 vs 高维、认知碾压、资源掠夺
- **人机冲突**：AI 觉醒、算法管控、仿生人对立
- **自我身份**：义体改造、意识数字化、时空后认知崩塌

---

## 群像与文明表

- 人类：坚守人性者 / 技术至上理性者 / 反科技怀旧者
- 非人：AI、仿生义体人、外星族群、意识上传数字生命
- 团队职能：科研、战斗、领航、后勤、谈判、情报

---

## 职业化校验（大纲自检）

- [ ] 技术规则全文统一，无前后矛盾
- [ ] 每项技术有能耗、短板、使用限制
- [ ] 科幻概念驱动主线，非背景板
- [ ] 设定通过事件释放，非说明书堆砌
- [ ] 硬科幻术语适度；软科幻保留基础逻辑

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
