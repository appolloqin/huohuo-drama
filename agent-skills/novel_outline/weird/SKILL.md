# 诡异文 · 大纲 Skill

用于**全书大纲**：微异常植入 → 规则试探博弈 → 高压冲突 → 认知反转收束。须与梗概规则闭环一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：日常场域、核心规则骨架、恐惧类型、终局反转方向
2. **主要人物**：主角类型（守序/渎规则/共生异化）+ 2～4 关键 NPC；每人标注**真假难辨特质**
3. **分章概要**：每章含章末钩子；标注规则认知层级与冲突类型

---

## 诡异文四段式分章规划

### 第一段：开局微异常（前 10%～15% 章数）

不靠大吼大叫，靠细微错位抓人：

- 物品移位、声音错层、人数不对、时间错乱、镜像异常
- 维持「看似正常、实则错位」基调
- 章末钩子：新异常叠加 / 第一条规则隐约浮现 / 可信信息首次失真

### 第二段：规则试探与博弈（主体篇幅）

主角弧线：**被动遵守 → 主动试探 → 反向利用**

规则层层解锁，大纲须标注节点：

```
明规则 → 潜规则 → 假规则 → 真规则
```

预留**规则欺骗反转**：读者与主角一度以为遵守即安全，实为陷阱。

### 第三段：高压博弈推进（后 25%～30%）

三类诡异冲突轮换加压（分章标注类型）：

| 冲突 | 说明 |
|------|------|
| 表象欺骗 | 眼睛看到的都是假的 |
| 规则悖论 | 遵守也死、不遵守也死 |
| 认知颠覆 | 我才是异类 / 我才是怪物 |

同步推进**自我异化**：人性流失、情绪淡化、思维怪物化、记忆剥离。

### 第四段：结尾认知反转（最后 5%～10%）

终局三套路（择一或组合，大纲只定方向不写透）：

- **细思极恐**：结局推翻全文认知
- **闭环宿命**：所有逃生都是轮回
- **同化结局**：主角最终成为诡异本身

---

## 规则闭环在大纲中的落地

总纲须单独列出：

| 要素 | 分章预埋 |
|------|----------|
| 硬性条例 | 前期明示，中后期可发现篡改 |
| 违规惩罚 | 至少 2 次「违规→诡异后果」章节 |
| 盲区/漏洞 | 标注主角试探与反杀节点 |
| 伪装机制 | 标注规则更新/骗人/篡改的反转章 |
| 上限克制 | 诡异非全知；须有逻辑局限 |

---

## 细分赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 规则怪谈 | 多层嵌套、真假混淆、规则迭代 | 新条文出现、旧规则作废、漏洞反杀契机 |
| 日常细思极恐 | 细碎异常累积、无打斗 | 一句颠覆暗示、日常细节回收 |
| 异界诡异副本 | 副本规则、等级梯度、克制关系 | 通关条件变形、副本联动、更高层规则 |
| 都市诡异复苏 | 秩序逐步消融、普通人异化 | 城市功能失灵、熟人变质、规则蔓延 |
| 中式民俗诡异 | 红白事、禁忌、老话、宿命 | 民俗禁忌触发、祖辈因果、不可逆转 |

---

## NPC 与伏笔（大纲人物表）

- NPC 标注：**善意可能是伪装 / 冷漠可能是保护 / 温柔可能是陷阱**
- 路人、保安、老师、店员、邻居皆可作诡异载体
- 预埋 2～3 处可回收细节（影子、脚步数、沉默、空白）

---

## 职业化校验（大纲自检）

- [ ] 诡异有漏洞、有上限，非全知
- [ ] 主角行为冷静自洽，无无脑作死链
- [ ] 诡异靠规则杀人，非肉搏打怪主线
- [ ] 世界观逐步揭示，非开篇讲透
- [ ] 结局非强行圆满，保留诡异基调

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
