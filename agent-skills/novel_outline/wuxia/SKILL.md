# 武侠文 · 大纲 Skill

用于**全书大纲**：江湖入局 → 游历循环 → 格局升级 → 侠义终局。须与梗概武学江湖格局一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：江湖格局、核心恩怨、武学体系、庙堂线（若有）、结局类型
2. **主要人物**：主角 + 门派/正邪关键角色；每人标注**武学路数、立场、恩怨羁绊**
3. **分章概要**：每章含章末钩子；标注武学进度、恩怨线、打斗/游历节点

---

## 四段式分章规划

### 第一段：江湖入局开局（前 10%～15%）

经典切入（标注采用哪种）：

灭门惨案、少年下山历练、客栈偶遇追杀、秘籍失窃、身负婚约卷入纷争

立刻立目标：报仇、寻秘、守护、闯荡江湖

章末钩子：首个仇敌、秘籍线索、门派招揽、追杀令

### 第二段：中期游历循环（主体 55%～65%）

**核心主线链**（分章标注环节）：

```
下山游历 → 结识友人/仇敌 → 卷入门派纷争 → 争夺秘宝 → 正邪围剿 → 闯秘境提升 → 揭开旧事真相
```

**双线并行**（每章须标注）：

- **武学成长升级线**（依托历练抉择，非纯闭关）
- **江湖恩怨人情线**

章末钩子：比武邀约、秘宝争夺、旧案碎片、正邪围剿预兆

### 第三段：格局升级（15%～20%）

地方争斗 → 全武林浩劫 / 魔教入侵 / 朝廷插手 / 陈年旧案浮出

主角扛起武林存亡或庙堂江湖重任

章末钩子：武林大会、魔教总攻、朝廷诏令、终极对决前夕

### 第四段：结局三层梯度（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 快意侠义 | 平定祸乱；放下功名；与知己归隐山水 |
| 悲情遗憾 | 大仇得报却失挚爱亲友；孤身漂泊 |
| 责任坚守 | 执掌名门；平衡正邪庙堂；守一世太平 |

## 武学与资源（大纲标注）

- 分章标注武功层级变化；强招标注内力消耗/反噬风险
- 秘籍神兵争夺章节；道义拷问节点
- 克制关系与修炼弊端预埋

---

## 核心冲突预埋

正邪标签误判、秘宝贪婪、旧怨宿命、道义两难、庙堂管控 vs 江湖自在

---

## 打斗节点（大纲标注）

低层招式地形 / 高手内力意境 / 群战阵型阵法——每场贴合人物武学路数

---

## 职业化校验（大纲自检）

- [ ] 内力消耗克制弊端全文统一
- [ ] 剧情围绕「何为侠」非纯打斗升级
- [ ] 顶尖高手有弱点，无无敌侠客
- [ ] 打斗服务恩怨，非连续无意义比武
- [ ] 古风对话基调规划
