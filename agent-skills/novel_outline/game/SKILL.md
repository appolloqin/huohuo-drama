# 游戏文 · 大纲 Skill

用于**全书大纲**：开篇切入 → 副本/任务循环递进 → 势力升级 → 幕后真相与分层结局。须与梗概游戏闭环一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：游戏框架、核心机制、资源矛盾、幕后真相方向、结局类型
2. **主要人物**：主角天赋/身份 + 队友/对手；每人标注**职能分工与利益立场**
3. **分章概要**：每章含章末钩子；标注副本层级、系统解锁进度、爽点/压迫节奏

---

## 开篇切入（分章规划）

| 赛道 | 前 1～5 章要点 |
|------|----------------|
| 网游/全息网游 | 开服重生或新手上线；面板展示；新手任务；首波资源冲突 |
| 无限死亡游戏 | 强制拉入；系统规则明示；第一场新手副本高压 |
| 穿单机/穿书 | 绑定原角色；剧情偏离；隐藏危险分支触发 |
| 经营建造 | 领地初建；资源循环；首批居民/防御 |
| 电竞职业 | 首秀/入队；战术亮相；首场关键对局 |

章末钩子：稀有掉落暗示 / 淘汰危机 / 剧情崩坏点 / 公会或系统异动

---

## 中期循环递进（主体篇幅）

### 通用剧情链（分章标注环节）

```
接取任务/进入副本 → 探索破解机制 → 玩家/怪物冲突 → 搜集资源提升 → 解锁隐藏线索 → 结算奖励/惩罚
```

### 长线升级逻辑

```
单人开荒 → 组队抱团 → 公会势力争夺 → 跨区对抗 → 揭露游戏幕后真相
```

### 副本梯度（大纲须规划节点）

新手开荒 → 精英秘境 → 地狱团本 → 终极特殊副本

每档标注：难度跃升、奖励类型、新机制解锁

### 系统逐步解锁

不在开局放出全部设定；随剧情解锁天赋、阵营、经济链、隐藏副本

## 五大系统在大纲中的分布

| 系统 | 大纲预埋 |
|------|----------|
| 战斗 | 职业分支、克制关系、boss 阶段、PVP 节点 |
| 生活生产 | 产业链章节、物资缺口驱动冲突 |
| 交易经济 | 囤货、拍卖、黑市、通胀压力章节 |
| 天赋觉醒 | 稀有天赋获取、进化反噬节点 |
| 阵营声望 | 势力选择、领地争夺、NPC 好感转折 |

---

## 结尾分层设计（最后 5%～15%）

| 结局类型 | 适用赛道 |
|----------|----------|
| 网游爽文 | 全服第一、最强公会、掌控经济链 |
| 无限流悬疑 | 揭穿创造者、逃离、轮回、同化系统 |
| 温情和解 | 人与 AI NPC 双向救赎、打破残酷规则 |

须与代价体系、伏笔形成闭环。

---

## 细分赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 传统/全息网游 | 升级打怪、装备、公会战、竞技场、商业囤货 | 极品掉落、版本更新、跨服战预告 |
| 无限闯关死亡游戏 | 诡异/规则副本、人性博弈、高死亡率 | 淘汰倒计时、规则篡改、双结局分支 |
| 穿单机/穿书 | 原剧情对照、规避危机、改写命运 | NPC 觉醒、剧情崩坏连锁 |
| 经营建造 | 领地循环、建筑升级、外交贸易 | 入侵危机、资源断供、扩张抉择 |
| 电竞职业 | 团战战术、联赛竞争、战队磨合 | 关键局逆转、转会风波、决赛对决 |

---

## 冲突预埋（大纲标注）

- **资源冲突**：稀有装备、唯一任务、积分争夺
- **规则冲突**：系统漏洞、与 GM/系统对抗
- **人性冲突**：背叛、舍己、假意合作
- **世界观冲突**：平行世界真相、系统收割人类

---

## 群像配置

队友：输出、坦度、辅助、侦查、后勤商人、解谜智囊

对手：氪金土豪、阴险会长、独行猎杀者、伪善通关者、利己背叛者

NPC：商人、任务发布者、隐藏导师、自主意识 AI、伪装怪物

---

## 职业化校验（大纲自检）

- [ ] 等级/装备/战力梯度稳定，无前期小怪后期秒杀崩坏
- [ ] 惩罚、冷却、道具效果全文统一
- [ ] 强者有短板，通关伴随代价
- [ ] 赛道定位清晰，不杂糅违和元素
- [ ] 设定随剧情解锁，非开局倾倒

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
