# 校园文 · 大纲 Skill

用于**全书大纲**：开学初识 → 日常循环 → 升学转折 → 青春结局。须与梗概学段赛道一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：学段、赛道、男女主关系、核心压力、隔阂来源、结局类型
2. **主要人物**：男女主 + 同桌室友老师家长；每人标注**学业状态、情感立场、家庭类型**
3. **分章概要**：每章含章末钩子；标注学业/感情双线、季节节点、考试事件

---

## 四段式分章规划

### 第一段：开学初识开局（前 10%～15%）

四大切入（标注采用哪种）：分班、调座位、新宿舍、转学生、军训

- 男女主初次交集；心动伏笔
- 校园环境与权力分层亮相
- 章末钩子：首次微小互动、座位/宿舍关系、流言萌芽

### 第二段：日常循环成长（主体 55%～65%）

**校园核心框架**（分章标注环节）：

```
日常上课晚自习 → 考试排名压力 → 课间独处互动 → 运动会/艺术节 → 流言误会隔阂 → 私下和解暧昧升温 → 家庭/升学压力
```

**双线并行**（每章须标注）：

- **青春感情线**
- **学业成长主线**——感情依附日常，不空谈恋爱

章末钩子：月考排名、流言爆发、家长会、微小糖点、隔阂加深

### 第三段：升学关键转折（15%～20%）

文理分科、高三誓师、艺考集训、高考冲刺、毕业倒计时

升学压力放大矛盾；决定分离或并肩

章末钩子：分科结果、艺考校考、志愿抉择、短暂分离危机

### 第四段：结局三层梯度（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 圆满青春 | 同城市大学、双向奔赴、长久约定 |
| 遗憾意难平 | 毕业异地、暗恋止于夏天、错过成回忆 |
| 重生弥补 | 改写选择、不再怯懦、并肩理想院校 |

---

## 六大赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 甜宠暗恋 | 课间微糖、双向试探、克制青涩 | 微小破例、误会和解、雨天同行 |
| 现实虐文 | 家庭创伤、冷暴力、升学分歧 | 家长会、孤立、异地预兆 |
| 重生校园 | 避伤害、冲高考、靠近遗憾之人 | 记忆卡点、排名跃升、主动破冰 |
| 艺考竞赛 | 集训辛苦、高压竞争、共同奋斗 | 校考、比赛、失利或突破 |
| 悬疑霸凌 | 失踪流言、心理压迫、救赎 | 线索、对峙、和解或揭露 |
| 大学校园 | 社团寝室、实习、异地、求职 | 选课冲突、实习分离、毕业抉择 |

---

## 校园事件日历（大纲分布）

月考、期中期末、运动会、校庆、家长会、竞赛名额、联考——按季节分布避免同质重复

---

## 群像配置

同桌室友助攻、流言配角（温和不极端）、班主任、欢喜冤家、家长三类（施压/缺席/开明）

---

## 核心冲突预埋

学业排名分科艺考、人际造谣孤立、情感暗恋被发现、现实贫富反对异地

---

## 职业化校验（大纲自检）

- [ ] 学业主线不被人情完全掩盖
- [ ] 冲突贴合少年心智，非成人狗血
- [ ] 人设行为符合年龄
- [ ] 无悬浮财富特权
- [ ] 甜虐节奏与学段匹配

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
