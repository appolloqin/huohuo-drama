# 种田文 · 大纲 Skill

用于**全书大纲**：破穷开局 → 稳步崛起循环 → 家事轻冲突 → 安稳结局升级。须与梗概资源闭环一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：赛道、开局困境、生存链条、家业升级路径、结局梯度
2. **主要人物**：主角 + 家人分工 + 关键邻里；每人标注**职能与利益立场**
3. **分章概要**：每章含章末钩子；标注季节、资源进展、冲突类型、小收获点

---

## 四段式分章规划

### 第一段：开局破穷/破困（前 10%～15%）

- 直击痛点：家贫、欠债、饥荒、破屋、受欺、流离
- 快速立目标：吃饱、安稳、存粮、立家、立足
- 章末钩子：第一笔收入希望、分家契机、天灾预警、邻里关系初显

### 第二段：稳步崛起循环（主体 55%～65%）

**种田核心循环**（分章标注环节，每次微小升级）：

```
开荒改造 → 种植养殖 → 手工加工 → 摆摊变现 → 积累资产 → 改善生活 → 解决矛盾 → 升级家业
```

- 四季更替：春种夏耕秋收冬藏，剧情不重样
- 每轮循环须有**新变化**：新作物、新工艺、新销路、新资产
- 章末钩子：丰收预告、新生意、邻里风波、家庭小转折

### 第三段：轻冲突推进（穿插全篇，中后期加压）

生活化细碎冲突轮换：

分家纠纷、极品亲戚、邻里嫉妒、物价波动、天灾虫害、家底薄风险

**不靠大虐大杀**；冲突温和但不断节奏

### 第四段：结局升级（后 10%～15%）

| 梯度 | 大纲落点 |
|------|----------|
| 温饱安稳 | 衣食无忧、家人安康 |
| 家业兴旺 | 铺面产业、良田成片、家境富足 |
| 家族崛起 | 子弟读书出仕、立足一方 |
| 世外归隐 | 小院安稳、岁月静好、治愈圆满 |

---

## 资源链条（大纲标注）

分章标注涉及哪条链：

产出 / 增值 / 储蓄 / 风控——家业升级须可追溯

## 五大赛道大纲侧重

| 赛道 | 分章推进 | 章末钩子 |
|------|----------|----------|
| 古言传统 | 农耕时序、分家立户、市井摆摊、赋税节令 | 收成、新铺、亲戚上门、节令集市 |
| 现代返乡 | 小院改造、田园美食、松弛日常、远离内卷 | 第一批客人、网络订单、季节活动 |
| 末世硬核 | 开荒育种、基地扩建、物资循环、风险预判 | 虫害、掠夺预警、新作物成活 |
| 修仙种田 | 灵植成熟、洞府升级、丹材收获、药田扩张 | 灵兽、秘境种子、丹成 |
| 星际种田 | 星球生态、域外作物、基地模块、贸易航线 | 生态指标、商船、环境危机 |

## 家人群像（大纲人物表）

主心骨、劳力、巧手、智囊、孩童——分章标注各自成长线与分工变化

孩童可预留读书、打杂、未来希望章节

---

## 邻里人际（大纲预埋）

良善 / 市井 / 恶劣三类——温和冲突节点，非全员恶人

---

## 四季与细节（大纲自检）

- [ ] 春种夏耕秋收冬藏有分布，非重复同质章节
- [ ] 农时、季节、作物、手工流程逻辑一致
- [ ] 增益循序渐进，无暴富跳跃章节
- [ ] 无强行宫斗权谋玄幻大线破坏松弛感
- [ ] 善良有底线、帮扶有分寸

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
