# 网文写作 · 15项一致性 Skill（全书大纲）

## 核心总纲

> 剧情、人物、设定、状态只允许有铺垫的渐变，大纲阶段即须**锁死递进关系**，避免后章与前章矛盾。

---

## 分章概要附加标注（推荐）

每章概要末尾可加一致性标签（择要，有升级/战斗/换场时必填）：

```
环境：破庙夜雨→次日清晨官道
状态：炼气三层→炼气四层（秘境顿悟）
伤势：重伤→轻伤（服丹调息）
关系：与王长老敌对加深
伏笔回收：第3章玉佩异动
```

---

## 15项大纲预埋要点

| Skill | 大纲须避免 |
|-------|------------|
| 1 环境 | 无过渡换场景；昼夜乱跳 |
| 2 修为 | 后章境界低于前章无原因；连破大境无铺垫 |
| 3 资源 | 道具无限；破损自动复原 |
| 4～5 神态衣着 | 大战后下一章「衣冠楚楚」无交代 |
| 6 人设 | 性格无理由反转 |
| 7 台词 | 古风章出现现代梗（大纲勿规划） |
| 8 伤势 | 重伤下章满血无恢复章 |
| 9 时间 | 一章内完成不合理长途+大战+突破 |
| 10 人际 | 死敌突然同盟无剧情链 |
| 11 伏笔 | 埋了不收；规则前后矛盾 |
| 12 动作 | 束缚状态仍规划结印施法 |
| 13 认知 | 主角未调查却知幕后真相 |
| 14 功法 | 限定技能当常规手段滥用 |
| 15 情绪 | 灭门下一章谈笑风生 |

---

## 修为曲线（有升级体系时）

1. 全书主角核心修为**默认只升不降**；跌境章须标注原因
2. 大突破分布合理，每突破前有铺垫章
3. BOSS 登场战力 ≥ 当时主角或注明压制设定

---

## 大纲完成后自检

- [ ] 按章号梳理境界/关键状态，无无故下降
- [ ] 伏笔有回收节点
- [ ] 时间线、地点链连贯
- [ ] 终局状态高于开局（或悲剧路线合规回落）

---

## 口诀

> 无铺垫不变、有依据才变、规则绝不随便变。

---

## 输出约束

- 与总纲设定用语统一；分章概要格式不变
