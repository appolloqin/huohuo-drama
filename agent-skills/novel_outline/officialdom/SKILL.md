# 官场文 · 大纲 Skill

用于**全书大纲**：基层入局 → 中层博弈循环 → 高层对抗 → 分层结局。须与梗概层级权责与派系一致。

---

## 大纲通用结构

1. **总纲**（150～300 字）：层级起点、核心博弈资源、派系格局、民生主线、结局类型
2. **主要人物**：主角 + 上下级同僚关键角色；每人标注**派系归属、诉求、底线**
3. **分章概要**：每章含章末钩子；标注民生/博弈双线、资源争夺、话术冲突节点

---

## 四段式分章规划

### 第一段：基层入局开局（前 15%～20%）

- 初入体制/赴任；烂摊子、派系矛盾、历史积案、民生难题
- 认清规则、确立底线
- 章末钩子：首项民生难题、派系试探、上级态度初显

### 第二段：中层立足博弈循环（主体 50%～60%）

**核心剧情链**（分章标注环节）：

```
接手民生/项目难题 → 协调多方推进 → 触碰既得利益受阻 → 向上借力向下安抚 → 做出实绩获晋升资本 → 卷入更高派系斗争
```

**双线并行**（每章须标注）：

- **民生实干主线**：惠民政策、项目落地、积案化解
- **人事权力博弈支线**：站队、交换、汇报、人事调动暗示

章末钩子：项目卡点、信访爆发、人事异动、纪委关注、派系摊牌预兆

### 第三段：高层格局对抗（15%～20%）

- 政策制定、区域发展路线、顶层人事调整
- 权力诱惑与廉政风险；底线终极抉择
- 章末钩子：重大人事、反腐线索、路线对立、信任危机

### 第四段：结局分层（后 5%～10%）

| 类型 | 大纲落点 |
|------|----------|
| 实干圆满 | 坚守初心、平衡各方、惠民落地、稳步晋升 |
| 遗憾妥协 | 部分让步、守住底线但锐气磨灭 |
| 悲情警醒 | 排挤外放、辞官归隐、对抗贪腐身陷险境 |

## 关系网（大纲人物表）

对上：领会意图、汇报分寸、分忧不邀功

平级：制衡、利益交换、表面和气暗中博弈

对下：恩威并施、防欺上瞒下

标注关键人事调动侧面暗示权力斗争

---

## 核心冲突预埋

民生 vs 利益、派系路线对立、人事晋升竞争、底线抉择、上下级信任与信息差

---

## 对话与氛围节点（大纲标注）

公开会议话术 / 私下办公室 / 酒局应酬——三种语言风格章节分布

现代：会议室文件、下乡土路、信访大厅

古代：朝堂礼制、官衙文书、门生拜谒

---

## 职业化校验（大纲自检）

- [ ] 职级权责流程符合体制/朝堂逻辑
- [ ] 晋升有实绩、机遇、上层支持，非纯热血开挂
- [ ] 反派有完整诉求逻辑，非为坏而坏
- [ ] 民生主线不被人事博弈完全掩盖
- [ ] 反腐遵循法治流程，无越级乱查私刑桥段
