# 末世文 · 大纲 Skill

用于**全书大纲**：开局爆点 → 中期生存链循环 → 后期格局升级。须与梗概灾变体系一致。

---

## 大纲通用结构

1. **总纲**：灾变闭环、资源枷锁、势力格局、终极真相/结局方向
2. **主要人物**：主角 + 团队/敌对势力；每人**生存立场 + 软肋 + 资源诉求**
3. **分章概要**：每章含章末钩子；标注危机层级（外部/内部/自我）

---

## 黄金三段式分章规划

### 第一段：末日开局（前 10%～15% 章数）

- **无铺垫急开局**：灾变瞬间、秩序崩塌、第一波死亡冲击
- 快速建立压迫：混乱、伤亡、失联、资源断裂、人性暴露
- 章末钩子：更大威胁露头 / 信任首次破裂 / 刚需资源告罄

### 第二段：中期生存推进（主体篇幅）

**标准剧情链循环**（可多次变体，层层加压）：

```
搜集物资 → 遭遇危险 → 人际博弈 → 取舍牺牲 → 能力迭代 → 环境恶化
```

加压手段轮换：怪物变强、天气恶化、资源枯竭、内鬼危机、外部势力入侵

分章须标注本轮循环处于哪一步。

### 第三段：后期格局升级（后 20%～25%）

升级路径（择一或组合）：

```
小团队生存 → 据点争夺 → 势力博弈 → 区域规则 → 末世真相揭露
```

结局升维可选：个人求生圆满 / 团队立国 / 文明存续抉择

---

## 团队群像配置（大纲人物表）

末世标准职能至少覆盖 3 项：**战力 / 侦查 / 医疗 / 后勤 / 技术 / 指挥**

大纲须预留：团队摩擦、利益分歧、信任崩塌、抱团取暖的章节节点。

---

## 三重冲突叠加（高级大纲标注）

重要章节同时标注三类冲突（至少两类）：

| 类型 | 示例 |
|------|------|
| 外部 | 尸潮、天灾、敌对势力、规则异变 |
| 内部 | 猜忌、资源不足、立场分歧、内鬼 |
| 自我 | 底线动摇、恐惧、异化、异能代价反噬 |

---

## 细分题材大纲侧重

- **硬核求生**：分章强调物资账目、路线取舍、狼狈细节
- **异能进化**：标注战力阶梯、团战节点、势力版图变化
- **规则诡异**：伏笔预埋、规则逐步显形、真相反转节点
- **基地建设**：据点等级、制度里程碑、攻防战章节
- **末世言情**：规则/灾难服务情感；绝境偏爱、生死抉择与羁绊同步推进

---

## 职业化校验（大纲自检）

- [ ] 资源有储量上限，无凭空变物章节
- [ ] 怪物/环境/异能同步迭代计划
- [ ] 无圣母崩盘情节（危机时行为逻辑统一）
- [ ] 秩序崩塌后社会逻辑不回弹为温情都市
- [ ] 人性「非黑即灰」，行为服务于生存

---

## 输出约束

- 简体中文；「总纲」「主要人物」「分章概要」
- 格式：`第N章：标题 / 概要`；不要前言套话
