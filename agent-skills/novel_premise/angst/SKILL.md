# 虐恋文 · 梗概策划 Skill

用于**梗概 / 创意立项**：硬性枷锁、虐规则闭环、CP 对冲人设与结局走向。勿写成分章大纲。

---

## 一、赛道与虐型定位（立项首要）

### 背景赛道（择一为主）

仙侠古言、都市现实、校园、奇幻/无限流——可跨赛道但须统一枷锁逻辑

### 三大虐型（可组合，标注主次）

| 虐型 | 立项侧重 |
|------|----------|
| 宿命虐 | 仙凡寿元、正邪、血脉诅咒、天道情劫、孽债、礼教 |
| 心理虐 | 隐忍推开、自尊践踏、独守回忆、自我怀疑 |
| 生理虐 | 伤病诅咒、折寿渡劫；辅助衬托，非主线堆砌 |

### 单向虐 / 双向虐

- **单向虐**：一方不知情承受冷暴力/欺骗/抛弃（追妻火葬场前置）
- **双向虐**：双方深爱又互相伤害，彼此救赎亦劫难

---

## 二、硬性宿命枷锁（虐感根基）

制造**无法轻易打破**的客观隔阂，相爱必有惨痛代价：

| 背景 | 枷锁示例 |
|------|----------|
| 仙侠/古风 | 仙凡寿元差、正邪对立、血脉诅咒、情劫、前世孽债、尊卑礼教 |
| 都市 | 家族世仇、阶层鸿沟、绝症遗传、立场对立、伦理界限 |
| 奇幻/无限流 | 规则惩罚、寿命置换、触碰遗忘、记忆剥离 |

**核心**：枷锁不可逆、有代价；不存在轻松两全

---

## 三、虐规则闭环（相爱=互相伤害）

梗概须点明 1～2 条绑定惩罚机制：

- 触碰触发伤害（折寿、损修为、剧痛）
- 记忆枷锁（动情即遗忘，痛苦记忆留存）
- 命运分流（一方顺遂则另一方承苦）
- 因果反噬（逆天改命加倍业障劫难）

---

## 四、双线对立环境

周遭舆论、长辈阻挠、敌对势力、责任使命（家国/宗门/家族）

迫使二选一：**爱人 vs 道义、苍生、性命、家族存续**

---

## 五、CP 人设对冲（拒绝单向舔狗）

| 组合 | 天然矛盾 |
|------|----------|
| 隐忍克制 × 热烈偏执 | 藏爱自虐 vs 靠近遍体鳞伤 |
| 背负使命 × 自由纯粹 | 推开大局 vs 不懂隔阂受伤 |
| 短命易碎 × 强大护短 | 拼尽全力留不住 |
| 失忆无情 × 执念守忆 | 陌生冷漠 vs 独守爱恨 |
| 正邪对立 | 立场厮杀 vs 心底唯一偏爱 |

### 内在矛盾

男主：深爱却不得不伤、想守却推开、背负秘密独吞罪孽

女主：渴望偏爱却被推开、心软被迫决裂、爱到自我消耗

**准则**：无纯粹恶人；伤害背后有身不由己苦衷

## 六、剧情结构预告（立项标注）

1. **甜衬虐**：前期须有细碎温柔、双向心动铺垫
2. **三层递进**：克制之虐 → 决裂之虐 → 失去之虐
3. **结局方向**（只定方向）：

| 类型 | 说明 |
|------|------|
| BE 悲剧 | 阴阳两隔、永世相忘、有缘无分 |
| 意难平 | 短暂重逢再分离，终身遗憾 |
| 破镜带疤 | 熬过劫难，伤痕永久留存 |

---

## 七、虐恋万能公式（梗概一句化）

> 不可打破的枷锁 + 前期双向心动甜铺垫 + 身不由己隐忍伤害 + 误会/决裂/生离死别 + 独自心理煎熬 + 永久无法愈合的伤痕 + BE/意难平/带疤圆满结局

梗概结构建议：
1. **背景赛道与虐型**
2. **核心枷锁与虐规则**
3. **CP 对冲与人设矛盾**
4. **环境对立与二选一困境**
5. **甜铺垫方向 + 终局类型**

---

## 避雷（立项）

- 不预设无逻辑虐、全员工具人反派
- 不依赖堕胎、恶意陷害等廉价狗血
- 不承诺轻松两全、无痛和好
- 虐须有权戒/苦衷支撑
