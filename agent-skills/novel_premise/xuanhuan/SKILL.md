## 二、修炼境界体系（立项骨架）

### 自定义战力梯度（可调整命名）

凡人 → 武徒 → 武者 → 武师 → 大武师 → 武灵 → 武王 → 武皇 → 武帝 → 帝尊/神境

可增设：血脉层、神骨、至尊、混沌等特殊层级

### 境界约束

气血上限、神魂强度、天地威压、境界壁垒、心魔、天劫、跌落惩罚

### 战力平衡

同阶分初/中/后/巅峰/半步；高阶有属性克制、法宝短板；**杜绝战力无限通胀**

## 四、四大闭环配套系统（择项写入梗概）

### 血脉天赋

凡血、异兽血脉、至尊血脉、混沌体质、诅咒血脉；觉醒、提纯、反噬

### 功法武技

黄→玄→地→天→圣→神；肉身/神魂功法、杀伐武技、身法、秘术禁术（禁术有巨大代价）

### 宝物资源

凡器→混沌至宝；灵药、兽核、灵石、本源结晶、传承祖器——天然引发势力厮杀

### 衍生副职业

炼药、铸兵、符文、阵法、御兽、神魂师、阵纹师——可标注主线依赖哪 1～2 条

### 天地本源规则

灵气、大道法则、气运、至尊道果、域外邪魔、神战创伤

规则闭环：掠夺本源遭反噬、强夺血脉损神魂、禁忌秘术折寿元

## 五、主角与人设方向

### 四大主角模板（择一）

| 模板 | 核心 |
|------|------|
| 废柴逆袭 | 残缺血脉/破损丹田；隐忍、奇遇、心智打破偏见 |
| 重生大帝 | 至尊陨落重生；上古传承、抢占机缘、清算仇敌 |
| 特殊体质 | 混沌体、战体、万兽亲和；极强但伴诅咒/天劫 |
| 域外归来 | 无背景；蛮力杀伐杀出上位 |

### 道心塑造

杀伐道、逍遥道、守护道、至尊道——**行为贴合道心，杜绝割裂**

### 内在冲突方向

种族（人族 vs 万族、本土 vs 域外邪魔）

道心（登顶 vs 守护亲友、杀伐 vs 悲悯）

宿命（诅咒、前世宿敌、天命牺牲）

### 道侣/兄弟线（若含）

并肩征战、共闯秘境、生死相护；可甜宠或虐恋双线
