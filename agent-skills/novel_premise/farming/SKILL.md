# 种田文 · 梗概策划 Skill

用于**梗概 / 创意立项**：生存闭环、资源链条、主角定型与安稳结局方向。勿写成分章大纲。

---

## 一、赛道定位（立项首要）

| 赛道 | 立项侧重 |
|------|----------|
| 古言传统种田 | 脱贫致富、分家立户、农耕时序、市井生意；烟火朴实 |
| 现代返乡种田 | 城市逃离、小院改造、田园美食、慢生活解压 |
| 末世硬核种田 | 绝境开荒、育种保种、物资循环、基地自建 |
| 修仙种田 | 灵植培育、洞府升级、丹材药田、灵气养殖 |
| 星际种田 | 星球开荒、域外作物、生态重建、星际贸易 |

**基调**：低高压、慢节奏、稳步增益、闭环生存

梗概须点明：① 赛道 ② 开局困境 ③ 核心生存链条

## 二、安稳/灾后生存闭环

### 完整生存链条

吃、穿、住、用、行、安全、储蓄、资产、人脉

### 四大基底困境（择一）

| 基底 | 开局痛点 |
|------|----------|
| 古代贫苦 | 战乱流民、家徒四壁、赋税繁重、资源极缺 |
| 现代治愈 | 返乡归隐、小院破败、需经营重建 |
| 末世绝境 | 物资枯竭、环境恶化、靠种植养殖续命 |
| 修仙/星际 | 洞府/星球荒芜、需开荒培育 |

## 三、资源循环体系（种田第一核心）

### 四大链条（梗概须点明主线依赖哪几条）

| 链条 | 内容 |
|------|------|
| 产出链 | 耕种、采摘、养殖、手工、织造、烹饪、酿造、手作 |
| 增值链 | 加工、改良、提纯、包装、摆摊、代销、长期供货 |
| 储蓄链 | 存粮、存钱、存种子、存物资、存人脉、存安全资产 |
| 风控链 | 防灾、防旱、防涝、防病、防骗、防邻里掠夺 |

**核心**：自给自足闭环；循序渐进，一分耕耘一分收获

## 五、四段式节奏预告（立项标注）

1. **开局破穷/破困**：家贫欠债饥荒破屋受欺；目标吃饱安稳存粮立家
2. **中期稳步崛起**：开荒→种植养殖→加工变现→积累资产（循环升级）
3. **轻冲突推进**：分家、极品亲戚、邻里嫉妒、天灾虫害、物价波动
4. **结局梯度**（只定方向）：温饱安稳 → 家业兴旺 → 家族崛起 → 世外归隐
