## 三、武学体系闭环（武侠核心）

### 三层梯度

基础拳脚兵刃 → 上乘门派绝学 → 绝世秘籍

兵器：剑刀枪棍暗器软兵

### 内力经脉

丹田内力、经脉运行、心法、真气反噬、走火入魔；强招耗内力，无无限爆发

### 衍生体系

轻功、点穴、毒术、医术、暗器、阵法、易容、音律武学——存在克制

### 秘籍约束

苛刻条件、残缺弊端、心魔代价；杜绝人人轻松天下第一

## 七、四段式主线（立项标注）

1. **江湖入局**：灭门/下山/客栈追杀/秘籍失窃/婚约纷争
2. **游历循环**：游历→结识友敌→门派纷争→夺秘宝→正邪围剿→秘境提升→揭开旧事
3. **格局升级**：全武林浩劫、魔教入侵、朝廷插手、陈年旧案
4. **终局方向**：快意归隐 / 悲情漂泊 / 责任守太平

**双线**：武学成长 + 江湖恩怨人情；武功依托历练抉择
