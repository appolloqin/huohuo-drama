## 二、修仙境界体系（立项骨架）

### 标准链条（可自定义细分）

凡人 → 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫

### 境界约束

寿元上限、修为压制、境界心魔、渡劫（雷劫/情劫/业劫）、跌落惩罚

### 战力平衡

同阶有强弱差；高阶有克制短板；**杜绝无上限战力膨胀**

## 四、配套闭环系统（择项写入梗概）

### 灵根资质

单灵根、杂灵根、变异灵根、废灵根、先天道体——天赋差距与修炼利弊

### 功法法宝

凡器→法器→灵器→宝器→仙器→神器；正邪功法、残缺上古传承

### 资源循环

灵石、灵草、丹药、妖丹、天材地宝、传承玉简——天然争夺冲突

### 衍生副职业

丹道、器道、符道、阵道、御兽、剑道、音律——可标注主线依赖哪 1～2 条

### 天道因果

业力、因果、气运、心魔、天劫、轮回——行为对应奖惩（业障/功德/反噬）

## 五、主角与人设方向

### 四大主角模板（择一）

| 模板 | 核心 |
|------|------|
| 草根逆袭 | 凡人身、废灵根；隐忍、机缘、心智崛起 |
| 天骄重生 | 前世大能陨落；记忆避劫、改写宿命 |

### 道心设定（行为须贴合）

无情道、有情道、逍遥道、杀伐道等——**杜绝人设与道心割裂**

### 仙侣 CP（若含感情线）

道侣羁绊、双修、渡劫共担；适配甜宠/虐恋/势均力敌

### 内在冲突方向

道心冲突（长生 vs 情爱、杀伐 vs 善念、顺天 vs 逆天）

种族冲突（人妖、仙魔、寿元鸿沟）

宿命冲突（天命、轮回、前世恩怨）

```

```

## 避雷（立项）

- 不承诺无上限战力、快速跳境界
- 不预设道心与人设矛盾（无情道沉溺情爱等）
- 古言仙侠忌纯无脑升级；传统流忌弱化境界体系
- 仙门对话基调古风，非现代口语
