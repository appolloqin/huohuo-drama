## 二、异能起源闭环

### 主流起源（择一或组合）

### 觉醒规则

阶段：未觉醒 → 初阶 → 进阶 → 巅峰 → 异化

### 路线

**小众异能**：极少数人隐藏于普通人中

**全民异能**：世界规则变更，人人觉醒

## 六、主角与人设方向

### 四大模板（择一）

| 模板 | 核心 |
|------|------|
| 弱小特殊异能 | 看似无用、挖掘上限、智谋反杀 |
| 高危毁灭异能 | 战力顶尖、反噬极强、压抑异化 |
| 隐藏苟分流 | 顶级能力伪装弱异能、躲避抓捕 |
| 实验体异能者 | 人造改造、身体缺陷、寻自我身份 |

**要点**：异能贴合性格（暴躁→火焰爆破；隐忍→治愈植物）

### 内心冲突方向

身份割裂、异化恐惧、族群矛盾、道德抉择（秩序 vs 随心所欲）

## 七、四段式主线（立项标注）

1. **觉醒危机**：被动失控 or 先天隐藏被打破；目标隐藏/掌控/寻同类/抗迫害
2. **成长循环**：掌控→冲突→队友→秘境资源→对抗反派→进阶+起源秘密
3. **格局升级**：派系博弈、地下战争、世界真相、灾难源头
4. **终局方向**：守护共存 / 悲壮牺牲 / 归隐留白 / 颠覆暗黑
