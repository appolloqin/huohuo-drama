# 诡异文 · 梗概策划 Skill

用于**梗概 / 创意立项**：日常异化场域、规则闭环、未知恐惧与认知反转方向。勿写成分章大纲。

---

## 一、诡异世界观（立项核心）

### 日常异化公式

> **普通生活场景 + 不合常理的微小异常**

常见场域：学校、小区、夜班、公交、医院、楼道、便利店、出租屋

梗概须点明：① 哪个日常场景 ② 哪类微小错位（物品/声音/人数/时间/镜像）③「看似正常、实则错位」的基调

### 恐惧类型定位

**未知大于血腥**：不依赖 jump scare、不堆血腥

梗概卖点偏向：心理恐惧 / 逻辑恐惧 / 未知恐惧 / 认知恐惧

叙事原则：**越解释越少、越未知越恐怖**（立项只暗示，不揭穿）

---

## 二、诡异规则闭环（最重要）

梗概须写清规则骨架（可保留盲区）：

| 要素 | 说明 |
|------|------|
| 硬性条例 | 必须遵守的明文/默示规则 |
| 违规惩罚 | 触发后的诡异后果（非单纯肉体伤害） |
| 盲区/灰色地带 | 规则漏洞、未知区间 |
| 伪装机制 | 规则会骗人、更新、篡改 |
| 上限与克制 | 诡异能力边界；非全知 |

**禁忌预告**：诡异靠规则杀人，不靠肉搏；须有漏洞与局限。

---

## 三、主角人设方向

三类顶级诡异主人设（择一或组合）：

| 类型 | 特质 |
|------|------|
| 守序求生型 | 冷静刻板、严守规则、极度理智 |
| 渎规则型 | 试探漏洞、利用诡异逻辑反杀、高智商疯批 |
| 共生异化型 | 自身带诡异、半人半怪、同化代价、随时失控 |

心智基调：观察力强、细节敏感、情绪稳定、不乱作死、**怀疑一切、只信规则**

---

## 四、细分赛道定位

| 赛道 | 梗概侧重 |
|------|----------|
| 规则怪谈 | 多层规则嵌套、真假混淆、逻辑博弈 |
| 日常细思极恐 | 生活化细碎恐怖、无打斗、一句颠覆 |
| 异界诡异副本 | 多副本规则、等级梯度、闯关解谜 |
| 都市诡异复苏 | 现实逐步崩坏、普通人异化、秩序消融 |
| 中式民俗诡异 | 红白事、禁忌、老话、阴冷宿命感 |

---

## 五、诡异万能公式（梗概一句化）

> 正常日常 + 细微错位异常 + 真假嵌套规则 + 两难生存博弈 + 认知欺骗 + 自我异化代价 + 终局颠覆反转

梗概结构建议：
1. **场景与微异常**
2. **核心规则（含一条悖论或盲区）**
3. **主角类型与生存思维**
4. **赛道与恐惧类型**
5. **终局反转方向暗示**（细思极恐 / 闭环宿命 / 同化——勿写死细节）

## 避雷（立项）

- 不承诺「全知诡异」、一次性讲透世界观
- 不预设无脑作死主角、强行圆满结局
- 不把诡异写成普通打怪怪兽片
