---

## 一、赛道定位（立项首要）

| 赛道 | 立项侧重 |
|------|----------|
| 今穿古（历史/架空） | 现代知识降维；制度不适、思想冲突、低调发育 |
| 古穿今 | 古人三观碰撞现代；反差萌、学习烟火、高冷沦陷 |
| 穿书穿剧 | 剧情预知、改命、规避炮灰；剧情偏差崩坏 |
| 反向穿越 | 古代大佬穿现代；权谋思维吊打职场商界 |
| 跨界穿越 | 穿末世/玄幻/游戏；信息差+生存短板、逻辑破局 |

梗概须点明：① 穿越方向与落点世界 ② 触发机制与代价 ③ 主角定型

---

## 二、穿越规则闭环（根基）

### 三要素：规则 + 代价 + 限制

**触发机制**：意外、身死、系统绑定、执念共振、时空裂缝、书本完结

**身份落点**：原主身死、昏迷、魂穿替换、双魂共存、附身留白

**代价约束**（防无敌悬浮）：

记忆残缺、时空排斥、寿命折损、天道压制、不能随意改命、改动历史触发反噬

---

## 三、新旧认知反差（最大看点）

| 对比 | 内容 |
|------|------|
| 现代 vs 古代 | 知识、三观、法制、平等 vs 礼教、等级、愚昧、强权 |
| 未来 vs 原始 | 科技思维 vs 落后世界观 |
| 穿书 | 上帝视角预知 vs 书中人物未知命运 |

---

## 四、世界基底适配

- **历史穿越**：朝代制度、官职、民俗、赋税、战争节点贴合时代
- **架空古穿**：自创王朝但礼法阶级经济自洽
- **穿书**：原剧情线、人设、节点完整；可制造偏差
- **跨界**：信息差优势 + 新世界规则生存短板

---

## 五、主角人设方向

### 现代人灵魂落地

保留现代三观、思维、底线；初期水土不服；成长：现代思维→适配乱世→外圆内方守本心

### 五大爆款定型（择一）

| 类型 | 核心 |
|------|------|
| 理智逆袭型 | 现代知识逻辑眼光降维；冷静稳发育 |
| 穿书自救型 | 剧情预知；规避死局改炮灰命运 |
| 权谋翻盘型 | 洞察人心制度漏洞；借势上位 |
| 实业强国型 | 基建经济民生技术革新 |
| 苟稳发育型 | 低调藏拙避锋芒；先活再崛起 |

### 原主遗留矛盾（开局抓手）

恶毒名声、欠债、结仇、婚约、家暴、被厌弃、蠢事遗留——接手逆风翻盘

## 六、四段式主线（立项标注）

1. **落地危机**：穿来即困境；目标活下去站稳改命
2. **降维发育**：信息差→解危机→打脸→积累资本→引关注猜忌→更高博弈
3. **剧情偏差**（穿书）：改小命运→牵动主线→剧情偏移→未知危机→原创线
4. **终局方向**：凡人圆满 / 格局逆袭 / 时空闭环（回归/留守/遗忘）

---

## 七、穿越万能公式（梗概一句化）

> 现代灵魂落地旧世界 + 原主绝境开局 + 认知信息降维优势 + 时代规则束缚代价 + 逆风翻盘打脸 + 三观碰撞冲突 + 改写宿命结局

1. **赛道与穿越规则**

## 避雷（立项）

- 不承诺现代知识万能、无代价开挂
- 不预设穿越即古人化、高调作死无后果
- 穿书须预留剧情崩坏原创线
- 历史穿越忌篡改基础制度重大事件逻辑
