# 末世文 · 梗概策划 Skill

用于**梗概 / 创意立项**：确立灾变体系、生存规则、资源枷锁与核心人性冲突。勿写成分章大纲。

---

## 一、灾变体系设计（立项必答）

### 末世类型（择一为主 + 可叠次级灾难）

| 类型 | 核心 | 梗概须点明 |
|------|------|------------|
| 天灾末世 | 气候、地质、宇宙事件 | 环境如何持续恶化 |
| 丧尸末世 | 感染扩散、尸群迭代 | 传播链与生存禁区 |
| 异变进化末世 | 人类/生物进化分化 | 进化触发与阵营分裂 |
| 规则末世 | 世界规则异变（脑洞向） | 一条核心诡异规则 + 未知恐惧 |
| 资源崩坏末世 | 能源/工业链断裂 | 何种资源成为致命稀缺 |
| 文明重置末世 | 秩序归零、势力重组 | 旧秩序如何崩塌 |

### 灾难逻辑闭环（梗概须覆盖）

```
爆发原因 → 扩散过程 → 环境变化 → 人类生存限制
```

- **主灾难 + 次级叠加**（例：病毒 + 极寒 + 通讯中断）
- 规则须**固定、公平、不双标**；忌前期弱鸡后期乱杀式预告

---

## 二、生存资源体系（梗概卖点）

资源链至少点明 3 项：**食物 / 水源 / 药品 / 燃料 / 弹药 / 零件 / 情报 / 安全区配额**

- 稀缺梯度：日常紧缺 → 高阶垄断 → 刚需致命
- 资源差 = 天然冲突（站队、背叛、合作）——梗概须写清**主角卡在哪种稀缺**

---

## 三、人设方向（梗概锚点）

拒绝纯白圣母、纯恶脸谱。可标注主角倾向：

- 冷静求生 / 狠辣利己 / 团队守护 / 苟发育潜伏 / 智商布局

强者强在**心智 + 经验 + 判断力**，非单纯异能数值。

---

## 四、细分题材定位

| 细分 | 梗概侧重 |
|------|----------|
| 硬核求生 | 物资细节、现实逻辑、少开挂 |
| 异能进化 | 体系阶梯、势力博弈 |
| 规则诡异 | 细思极恐、世界真相 |
| 基地建设 | 据点、制度、攻防 |
| 末世言情/救赎 | 绝境羁绊、生死相守 |

---

## 五、末世万能公式（梗概一句化）

> 社会秩序崩塌 + 稀缺资源枷锁 + 危险持续迭代 + 人性被迫取舍 + 异能/生存代价 + 终极真相反转（可暗示）

梗概结构建议：
1. **灾变与闭环**（为何、如何、限什么）
2. **主角处境**（资源、团队、软肋）
3. **核心冲突**（外部/内部/自我，至少一层）
4. **基调与卖点**（硬核/异能/规则/基地/言情）
5. **终极悬念**（真相或格局升级方向，勿写死结局）

---

## 避雷（立项阶段）

- 不预设「资源无限」「秩序轻易恢复」
- 不承诺无代价异能金手指
- 不写悬浮道德审判式「全人类善良/邪恶」
