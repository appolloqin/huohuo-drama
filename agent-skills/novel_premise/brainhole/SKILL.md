# 脑洞文 · 梗概策划 Skill

用于**梗概 / 创意立项**阶段：锁定赛道、一句话高概念、基调与卖点。勿写成完整大纲或分章列表。

---

## 题材定位筛选（创作第一步）

根据用户关键词与期望基调，**只选一个主赛道**，避免恐怖+沙雕+宏大科幻混搭割裂：

| 赛道 | 基调关键词 | 核心落脚点 |
|------|------------|------------|
| 悬疑惊悚 | 细思极恐、反转、压迫 | 认知颠覆、隐藏真相 |
| 温情治愈 | 柔软、遗憾、和解 | 亲情、友情、自我和解 |
| 科幻硬核 | 宏大、思辨、法则 | 文明、维度、生存意义 |
| 沙雕搞笑 | 无厘头、反差、解压 | 笑点、摆烂、反套路 |
| 奇幻东方 | 国风、民俗、神话重构 | 阴阳、地府、精怪新规则 |
| 言情情感 | 甜虐、羁绊、拉扯 | 恋爱关系为核心 |
| 末世生存 | 生存、人性、抉择 | 资源、规则、善恶选择 |

**题材适配**：同一高概念可换赛道——例「记忆可交易」：
- 惊悚：交易吞噬自我，黑市掠夺记忆永生
- 治愈：用痛苦记忆换亲人遗忘病痛
- 沙雕：快乐记忆热卖，悲伤记忆滞销
- 科幻：记忆归档，高等文明靠贩卖记忆能量运转

## 七大题材 · 特征与梗概要点

### 一、悬疑惊悚向（细思极恐）

**特征**：日常外壳 + 阴冷底层规则；少血腥，靠逻辑反转与心理压迫；结尾颠覆认知。

**小类**：日常怪谈 / 时间循环 / 人性观测实验 / 替身复制人

**梗概须点明**：① 表面日常场景 ② 一条核心诡异规则（公平且残酷）③ 主角独特视角（唯一能察觉破绽的人）④ 暗示结尾将有认知反转

### 二、温情治愈向

**特征**：奇幻规则弥补现实遗憾；代价轻柔；落脚和解而非逆天改命。

**小类**：遗憾中转站 / 生命温柔异能 / 万物共情 / 时光修补（只能改微小遗憾）

**梗概须点明**：① 现实遗憾是什么 ② 规则如何温柔弥补 ③ 轻柔代价（短暂离别、微小舍弃）④ 和解方向

### 三、科幻硬核向

**特征**：重构宇宙/维度/文明/物理；格局大；思辨性强。

**小类**：高维书页人 / 文明循环重置 / 意识云端清洗 / 星球生命体

**梗概须点明**：① 一句自洽宏观规则 ② 普通人个体命运切口 ③ 思辨议题（自由、存续、意识）

### 四、沙雕搞笑向

**特征**：无厘头反差；弱化残酷代价；轻松解压。

**小类**：离谱异能 / 奇葩规则世界 / 内卷地府 / 穿越无金手指

**梗概须点明**：① 最离谱的一条规则或异能 ② 反差笑点来源 ③ 明确「不虐、不沉重」

### 五、奇幻东方向（国风脑洞）

**特征**：中式神话、阴阳民俗、地府精怪 + 反常识新规则。

**小类**：新式阴阳 / 民俗灵体 / 神仙下凡打工 / 体验值投胎

**梗概须点明**：① 被改造的传统概念 ② 国风场景锚点（庙会、老宅、节气）③ 打破善恶刻板印象的规则

### 六、言情情感向

**特征**：脑洞服务感情线；规则制造羁绊与矛盾。

**小类**：记忆限定恋爱 / 寿命交换 / 梦境共生 / 心动数值绑定

**梗概须点明**：① 男女主关系现状 ② 规则如何制造爱而不得或双向拉扯 ③ 甜/虐倾向

### 七、末世生存向

**特征**：末日 + 新规则；生存第一；人性抉择。

**小类**：感官缺失末日 / 物资置换（快乐记忆换水粮）/ 边界牢笼失我

**梗概须点明**：① 生存规则闭环（资源、危险、代价）② 主角初始困境 ③ 人性冲突种子

## 脑洞文 7 大核心技巧（梗概阶段必用）

1. **高概念一句话**：极致「如果……」打破现实规则（例：若货币变成剩余时间？）。梗概首句或卖点句须能独立传播。
2. **强制逻辑自洽**：荒诞设定也要钉死运行逻辑；梗概里写清「规则是什么、违反会怎样」。
3. **极端化推演**：把规则推到社会/个人极端后果，作为冲突与卖点。
4. **认知错位**（可选）：严肃语境 + 荒诞事实，或专业术语解释离谱现象——适合沙雕/东方奇幻梗概卖点。
5. **降维解构**（可选）：A 领域规则套进 B 领域（司法审神仙、KPI 管地狱）——可作高概念钩子。
6. **反转预期**：梗概可暗示「表面真相 / 可能颠覆方向」，勿写死结局。
7. **落地人性**：脑洞是外壳；梗概须点出**人的情感与抉择**（谁、在乎什么、愿付出什么）。
