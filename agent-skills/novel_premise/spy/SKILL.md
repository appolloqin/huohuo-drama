## 二、三方势力制衡（谍战铁三角）

| 势力 | 要点 |
|------|------|
| 我方 | 潜伏人员、情报线、交通站、联络员、单线联系 |
| 敌方 | 特高课、宪兵队、情报科、审讯科、内鬼 |
| 中立灰色 | 军阀、商界、双面掮客、墙头草官员、平民 |

**核心**：没有绝对安全，人人互相试探

民国典型：军统 / 地下党 / 日军傀儡三足制衡

---

## 三、谍战规则闭环

### 单线联系

不横向接触、不留名单、不私藏证据

### 情报流程

获取 → 加密 → 传递 → 转接 → 销毁

### 暴露代价

全城搜捕、钓鱼执法、家人牵连、身份清算

### 潜伏底线

克制感情、隐藏锋芒、日常伪装、绝对隐忍

---

## 四、时代氛围（民国赛道）

宵禁、盘查、特务盯梢、街头监听、茶馆窃听、电报滴滴声、深夜搜查、物资管控、舆论严控

**基调**：表面平静繁华，内里步步杀机

---

## 五、主角与人设方向

### 四大模板（择一）

| 模板 | 核心 |
|------|------|
| 深海潜伏型 | 敌营高位；儒雅克制；每句话有算计；永远藏底牌 |
| 刀尖新人型 | 年轻联络员；谨慎青涩；高压成长；容错极低 |
| 双面摇摆型 | 表面效忠一方；内心效忠信仰；精神高压 |
| 悲情殉道型 | 背负骂名；牺牲情爱名誉；无人知晓功绩 |

### 对手立体化

高观察力、多疑、细节控、钓鱼、心理施压；忠于职责非纯粹坏

**高手对局**：比谁更能藏，非比谁更能打

### 内核冲突

信仰（苟活 vs 殉道）、身份（白天敌人深夜卧底）、情感（相爱不能认）、道德（牺牲无辜战友说谎）

---

## 六、四段式主线（立项标注）

1. **高压开局**：断联、交通站暴露、叛徒、全城排查、身边试探
2. **博弈循环**：任务→伪装执行→试探排查→叛徒钓鱼→真假情报→牺牲取舍→惊险过关→更大危机
3. **高潮破局**：三重反转（表层危机→中层内鬼→深层将计就计）
4. **终局方向**：隐忍圆满 / 悲壮殉道 / 意难平留白

---

## 七、谍战万能公式（梗概一句化）

> 三方高压制衡 + 潜伏伪装枷锁 + 真假情报博弈 + 试探钓鱼心理对局 + 取舍牺牲家国 + 局中局多重反转 + 隐忍/悲壮无名结局

梗概结构建议：
1. **赛道与时代背景**
2. **三方格局与初始危机**
3. **主角类型与潜伏身份**
4. **核心任务与信任/内鬼隐患**
5. **终局梯度方向**
