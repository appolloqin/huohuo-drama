## 一、赛道定位（立项首要）

| 赛道 | 立项侧重 |
|------|----------|
| 现代基层乡镇 | 三农、拆迁、信访维稳、招商、宗族矛盾；写实民生 |
| 高层城市政务 | 城市规划、重大项目、财政人事、纪委反腐、舆情；潜台词博弈 |
| 古代朝堂权谋 | 皇权相权、文武对立、科考党争、赈灾边防；礼仪森严 |
| 反腐扫黑官场 | 纪委办案、保护伞链条、证据博弈、廉政底线 |

**基调**：克制、压抑、体面之下暗流涌动；实干是晋升根基，博弈是生存手段

梗概须点明：① 现代 or 古代 ② 层级起点 ③ 细分赛道

## 二、层级权责体系（立项骨架）

### 现代官场链条

村/乡镇 → 区县 → 市 → 省 → 中央

须点明：一把手/二把手/分管副职、局办、纪委、组织部等权责边界

核心权限：人事、财政、项目、信访、督查

### 古代朝堂

州县、府道、六部、内阁/军机处、皇权；文武分治、御史言官、藩王、士族

### 规则闭环

晋升渠道、考核标准、换届周期、请示汇报、回避制度、问责处分

**禁忌预告**：行事符合层级规矩，无随意越级开挂

## 四、圈层派系生态

### 派系类型

本土派、空降派、老干部保守派、实干少壮派、商人利益圈

### 利益绑定

同乡、同门、老领导下属、姻亲、项目合作

派系无绝对善恶；诉求为自保、晋升、守住利益

### 缓冲角色

无欲老干部、纪检中立、无派系技术干部

---

## 五、主角与人设方向

### 主流主角（择一）

| 类型 | 特质 |
|------|------|
| 实干理想型 | 坚守底线、解基层实事；初期受排挤，靠实绩上升 |
| 隐忍蛰伏型 | 洞悉规则、外圆内方；不站队，借势成事 |
| 空降破局型 | 上级下放；冲击固化利益格局 |

### 关键配角类型

投机利己、守旧稳健老干部、清廉刚直监察（纪委/御史）

### 人物核心特质

语言克制、潜台词多；决策先评估风险舆论上级派系；双面性——无纯粹好坏

## 六、四段式主线（立项标注起点）

```
基层入局 → 中层立足博弈循环 → 高层格局对抗 → 初心抉择结局
```

梗概只写**入局困境 + 行事底线 + 终局方向**

### 结局方向（只定方向）

实干圆满 / 遗憾妥协 / 悲情警醒（排挤外放、辞官、身陷险境）

---

## 七、官场万能公式（梗概一句化）

> 完整层级权责体制 + 民生项目/人事晋升博弈载体 + 多方派系制衡 + 主角底线与规则拉扯 + 基层实事主线 + 隐晦话术暗斗 + 初心抉择式结局

梗概结构建议：
1. **赛道与层级起点**
2. **核心博弈资源与派系格局**
3. **主角类型与行事底线**
4. **民生难题与利益阻碍**
5. **结局基调方向**

## 避雷（立项）

- 不承诺无脑热血掀翻地方势力、随意越级
- 不预设脸谱化贪腐反派
- 博弈不脱离民生主线
- 反腐须遵循法治流程基调
