# 校园文 · 梗概策划 Skill

用于**梗概 / 创意立项**：校园层级场景、矛盾载体、人设定型与青春结局方向。勿写成分章大纲。

---

## 一、赛道定位（立项首要）

| 赛道 | 立项侧重 |
|------|----------|
| 校园甜宠暗恋 | 细腻暗恋、双向试探、课间微糖；克制青涩治愈 |
| 校园现实虐文 | 原生创伤、冷暴力、升学分歧、异地遗憾 |
| 重生校园 | 成年记忆避伤害、冲刺高考、弥补年少遗憾 |
| 艺考/竞赛校园 | 集训高压、共同奋斗中滋生感情 |
| 校园悬疑/霸凌 | 失踪流言、冷暴力；心理压迫与救赎 |
| 大学校园文 | 社团寝室、实习求职、异地；多一层现实压力 |

梗概须点明：① 初中/高中/大学 ② 赛道类型 ③ 年代氛围（旧式/现代）

## 二、校园场景生态（立项骨架）

### 校内场景

教室、晚自习、操场、小卖部、医务室、天台、图书馆、宿舍、画室实验室、教务处、校门口小巷

### 校外场景

夜市、奶茶店、补习机构、公交、公园、联考考场、车站

### 权力分层

班主任、任课老师、教导主任、学生会、班干部、普通学生、校外混混

规则：校纪校规、月考期中期末、升学压力

---

## 三、核心矛盾载体

考试排名、分班、文理分科、艺考体考、运动会校庆、家长会、早恋管控、调座、寄宿矛盾、流言、升学择校、竞赛名额

**原则**：冲突依托真实校园刚需，不脱离学生身份

## 四、时代氛围（择一）

**旧式青春**：mp3、手写情书、黑板报、辣条、自行车、纸质错题本

**现代校园**：智能手机、短视频、网课、外卖、社团直播、答题卡

**四季框架**：春运动会、夏晚风晚自习、秋月考、冬跨年期末

## 五、主角与人设方向

### 爆款定型（择一或组合）

| 类型 | 特质 |
|------|------|
| 清冷学神 | 寡言自律、年级第一；唯独对主角破例 |
| 阳光少年/元气少女 | 外向热心、人缘好；靠近内向主角 |
| 敏感内向普通生 | 中游、自卑细腻；易被流言伤害 |
| 叛逆吊车尾 | 不爱读书；内心缺关爱；对一人专一 |
| 艺考艺术生 | 集训压力、感性细腻、作息特殊 |
| 重生校园 | 成年记忆；避伤害、改升学、弥补遗憾 |

### 情感逻辑

青春期懵懂克制暗恋 / 双向明目张胆偏爱

### 少年内核

自尊心强、情绪敏感、易心动、不懂表达、怕流言、在意排名、渴望被理解

## 六、四段式主线（立项标注）

1. **开学初识**：分班调座宿舍军训转校；初次交集、心动伏笔
2. **日常循环**：上课考试→课间互动→运动会艺术节→流言隔阂→和解暧昧→家庭升学压力
3. **升学转折**：分科、誓师、艺考集训、高考冲刺、毕业倒计时
4. **结局方向**：圆满同校 / 遗憾异地 / 重生弥补

**双线**：青春感情线 + 学业成长主线，不可割裂

---

## 七、校园万能公式（梗概一句化）

> 真实校园场景 + 升学考试压力 + 青涩双向心动 + 流言家庭分班隔阂 + 课间四季温柔细节 + 共同成长逆袭 + 圆满/遗憾青春结局

梗概结构建议：
1. **学段与赛道**
2. **男女主人设与初次交集契机**
3. **核心压力与隔阂来源**
4. **时代氛围与季节基调**
5. **结局梯度方向**

---

## 避雷（立项）

- 不脱离校园逻辑全程弃学恋爱
- 不预设成人化狗血、极端犯罪、巨额特权
- 高中生行为符合年龄心智
- 早恋侧重懵懂激励，非过度成熟亲密
