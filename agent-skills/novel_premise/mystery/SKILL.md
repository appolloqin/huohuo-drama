# 悬疑文 · 梗概策划 Skill

用于**梗概 / 创意立项**：案件骨架、悬念铁律、主角类型与双层反转方向。勿写成分章大纲。

---

## 一、赛道定位（立项首要）

| 赛道 | 立项侧重 |
|------|----------|
| 刑侦悬疑 | 物证链、逻辑排除、时间线；无超自然硬核推理 |
| 社会派悬疑 | 人性、原生家庭、社会阴暗；凶手可同情、结局致郁反思 |
| 心理悬疑 | 记忆错乱、人格伪装、视角欺骗；「你看到的自己也是假的」 |
| 校园悬疑 | 冷暴力、流言、霸凌、失踪；青涩阴冷反差 |
| 都市细思极恐 | 日常异常、邻里诡异、认知颠覆；少血腥 |
| 本格/反转悬疑 | 伏笔回收、逻辑闭环、表层+深层双反转 |

梗概须点明：① 赛道 ② 悬疑基调（冷硬/心理/微恐）③ 开篇钩子类型

---

## 二、悬念三大铁律（立项骨架）

1. **伏笔可回收**：道具、台词、动作、环境异常——前期不起眼
2. **线索可验证**：读者与角色可沿线索推理
3. **真相可闭环**：反转有铺垫，非瞎编

### 隐藏信息手段

留白、视角盲区、时间断层——制造信息差

---

## 三、案件结构链条（梗概须含）

```
案发环境 → 现场异常 → 物证链条 → 时间线 → 人物关系网 → 动机推导 → 真相推翻 → 终极反转
```

**禁忌预告**：线索断裂、凭空真相、动机缺失

---

## 四、氛围基调（择一）

| 基调 | 特质 |
|------|------|
| 冷硬刑侦 | 理性、证据链、逻辑碾压、无鬼神 |
| 心理悬疑 | 扭曲人格、记忆谎言、自我欺骗 |
| 微恐氛围 | 日常诡异、细节不对劲、细思极恐 |

---

## 五、主角与凶手方向

### 主角三类（择一）

| 类型 | 核心 |
|------|------|
| 逻辑推理型 | 观察极致、细节控、时间线重组、微表情判谎 |
| 心理侧写型 | 共情罪犯心理、推演人格缺陷 |
| 创伤天才型 | 过往阴影、偏执失眠高敏感、执念破案 |

### 凶手立体化

**动机**：执念复仇、心理扭曲、自保误杀、极端正义

**特质**：外表普通合群温和、毫无存在感（越正常越吓人）

每一步布局有目的，每一次隐藏有理由

### 误导设计

假线索、假口供、假视角；证人说谎（自保隐瞒）、队友误判、路人盲点

---

## 六、四段式主线（立项标注）

1. **开局钩子**：尸体/异常/预言开局；300字锁读者
2. **迷雾叠加**：线索→锁定→推翻→二重迷雾→关系崩塌→再反转
3. **高潮拆解**：伏笔回收、时间线闭合、动机落地
4. **双层结局**：表层破案 + 深层反转（真凶另有其人/幕后推手/主角陷局）

---

## 七、悬疑万能公式（梗概一句化）

> 诡异疑点开局 + 多层迷雾线索嵌套 + 真假混杂证词 + 逻辑推翻递进 + 人性心理深层动机 + 全程伏笔回收 + 表层破案 + 深层双反转结局

梗概结构建议：
1. **赛道与基调**
2. **案发核心疑点与关系网**
3. **主角类型与调查优势**
4. **凶手动机方向（可隐藏）**
5. **表层真相 + 深层反转暗示**

---

## 避雷（立项）

- 不承诺机械降神、无效伏笔、强行反转
- 不预设全员降智衬托主角
- 刑侦须符合物证流程逻辑基调
