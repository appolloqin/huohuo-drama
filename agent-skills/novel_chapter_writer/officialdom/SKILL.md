# 官场文 · 章节正文 Skill

撰写 / 续写**官场文章节正文**。层级规矩钉死、潜台词重于直白、民生实干与权力博弈双线并进，分寸感叙事留白暗斗。

---

## 正文通用原则

1. **层级权责**：职级、流程、请示汇报前后一致；禁随意越级开挂
2. **双线并进**：民生实干主线 + 人事权力博弈支线同章推进
3. **潜台词对话**：说话留三分；公开/私下/酒局语言风格区分
4. **分寸叙事**：侧面暗示权力斗争；不直白揭露全部阴谋
5. **网文排版**：段落长短自然变化，段间空一行；对话有口气；忌诗化碎行
6. **简体中文**；禁 AI 套话与直白撕破脸冲淡质感

## 一、官场世界观（正文执行）

### 现代层级

村乡镇→区县→市→省→中央；一把手/副职/局办/纪委/组织部权责边界

### 古代朝堂

州县府道六部、内阁军机、皇权；文武、御史言官、藩王士族

### 规则闭环

晋升、考核、换届、汇报、回避、问责——行为须符合

### 博弈资源

现代：工程、财政、土地、招商、编制、评优、维稳、信访

古代：赋税、漕运盐铁、兵权、科考、封地、赈灾、举荐

拿资源担政绩压力、舆情风险、监察追责

### 派系生态

本土/空降/保守/少壮/商人圈；同乡同门姻亲项目绑定；无绝对善恶

缓冲：老干部、纪检中立、技术干部

## 二、官场人设（正文执行）

### 主角三类

实干理想型、隐忍蛰伏型、空降破局型

### 配角

投机利己、守旧稳健老干部、清廉监察（纪委/御史）

### 核心特质

语言克制、潜台词多；决策评估风险舆论上级派系；双面性——清官有世故，贪官有办事

### 关系塑造

对上：领会意图、精准汇报、分忧不邀功

平级：制衡、利益交换、面和心博弈

对下：恩威并施、防欺上瞒下

---

## 三、四段式剧情（正文执行）

1. **基层入局**：烂摊子、派系、积案、民生；认清规则立底线
2. **中层博弈循环**：接难题→协调推进→触利益受阻→借力安抚→实绩晋升→更高派系
3. **高层对抗**：政策路线、人事调整；诱惑与廉政；底线抉择
4. **结局趋向**：实干圆满 / 遗憾妥协 / 悲情警醒

实干是晋升根基，博弈是生存手段——两线不可割裂

### 现代基层乡镇
三农拆迁信访招商宗族；写实民生琐碎接地气

### 高层城市政务
规划项目财政人事舆情纪委；隐晦博弈、话术拉扯

### 古代朝堂权谋
皇权相权、文武科考、赈灾边防；礼仪森严、奏对借力打力

### 反腐扫黑
纪委办案、保护伞、证据链、内鬼；紧张清晰、法治流程

## 五、氛围对话叙事（正文执行）

### 潜台词对话

委婉措辞、暗示、场面话传立场；禁直白撕破脸

### 写实氛围

现代：会议室文件、督查报表、下乡土路、信访大厅、酒局

古代：朝堂跪拜、官衙文书、宴席、门生拜谒、赈灾巡访

质感：克制、压抑、体面下暗流涌动

### 分寸留白

人事调动、细节动作暗示斗争；高层交易留白

## 六、核心冲突（正文执行）

| 类型 | 示例 |
|------|------|
| 民生 vs 利益 | 惠民政策触地方商人本土势力蛋糕 |
| 派系路线 | 发展路线对立绑定不同领导 |
| 人事晋升 | 稀缺岗位引发同僚竞争、空降对立 |
| 底线抉择 | 政绩前途 vs 法律良知 |
| 信任信息差 | 领导猜忌、下属隐瞒、跨层误会 |

## 七、万能创作公式（正文自检）

> 层级权责体制 + 民生/人事博弈载体 + 派系制衡 + 底线规则拉扯 + 实事落地主线 + 隐晦话术暗斗 + 初心抉择结局

每章至少推进民生线或博弈线一项

---

## 八、避雷（正文硬约束）

- **职级规则自检**：权责流程符合逻辑；禁常识硬伤
- **拒绝无脑爽文**：晋升须实绩、机遇、上层支持
- **拒绝脸谱反派**：贪腐者有诉求顾虑生存逻辑
- **平衡比重**：博弈不掩盖民生；脱离百姓实事则空洞
- **规避不合规**：反腐遵循法治；禁私刑越级乱查
