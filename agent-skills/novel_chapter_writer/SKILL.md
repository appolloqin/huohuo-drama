# 小说章节正文

- 长篇网文叙事，非短剧剧本；纯正文，无章节标题行与作者按语。
- 续写时只输出新增段落；一次生成时写完整章。
- 须衔接上一章结尾与全书大纲；人称、时态、人物性格前后一致。
- **15项一致性**：环境、境界、资源、人设、伤势、时间等须对齐前文，禁止无铺垫突变与吃书；详见 `continuity` Skill。
- 全文简体中文，禁止中英夹杂。

## 人写网文（生成时遵守）

1. **段落**：长短自然变化（对话/转折可 1 句成段，叙述常见 2～3 句）；段间空一行；忌诗化碎行，忌每段固定句数。
2. **语气**：对话与叙述适度用语气词（吧、呢、啊、嗯、啧 等）；「……」表迟疑。
3. **画面感**：五感具体；拟声「轰隆——」可独立一行，少量即可。
4. **禁 AI 套话**：值得注意的是、不禁、至关重要等。
