# 脑洞文 · 章节正文 Skill

用于撰写 / 续写**脑洞文章节正文**。须紧接上下文，保持规则自洽与题材基调。

---

## 正文通用原则

1. **规则钉死**：已确立的世界规则前后一致；荒诞也要「一本正经」
2. **人性内核**：再飞的设定，落点在人的情感与选择
3. **网文排版**：段落长短自然变化，段间空一行；对话有口气；忌诗化碎行
4. **简体中文**；禁 AI 套话（值得注意的是、不禁、至关重要等）

---

## 七大核心写作技巧（正文执行）

| 技巧 | 正文执行 |
|------|----------|
| 高概念 | 开篇或本章首段尽快呼应核心「如果……」设定，勿偏离发动机 |
| 逻辑自洽 | 角色行为符合已公布规则；违规须有明确后果 |
| 极端推演 | 通过具体场景展现规则的极端后果（社会乱象、个人绝境） |
| 认知错位 | 严肃语境写荒诞事，或专业术语解释离谱现象——沙雕/东方奇幻尤适用 |
| 降维解构 | 现代制度/ KPI / 司法逻辑硬套神话地府修仙，制造体系碰撞 |
| 反转再反转 | 先建立认知再打破；反转须有前文伏笔；可多层（打败魔王→封印维持世界→黑化守平衡） |
| 落地人性 | 时间买卖打动人的是母亲为孩子买青春，不是机制本身 |

## 分题材正文专属 Skills

### 一、悬疑惊悚向

**伏笔预埋**：散落微小异常（习惯改变、物品错位、他人遗忘的细节）；后期回收时须对得上。

**克制留白叙事**：不完整揭露全部规则；保留未知恐惧感。

**日常异化塑造**：吃饭、上班、睡觉、社交赋予诡异约束，外表尽量正常。

**冷逻辑设计**：规则公平但残酷，无善恶偏向；放大无力感而非血腥。

**禁忌**：Jump scare 式血腥；一次性倾倒全部真相。

### 二、温情治愈向

**低伤害代价**：牺牲不致命——短暂离别、微小舍弃、一段记忆的淡出。

**情感羁绊刻画**：未说出口的话、旧物、陪伴、重逢；细腻温柔。

**和解式结局倾向**：章内可有小释怀；不追求逆天改命。

### 三、科幻硬核向

**自洽宏观规则**：物理/文明设定不前后打架；技术描写服务于法则。

**大小视角结合**：宏大背景下写具体人的恐惧、希望、选择。

**思辨内核**：轻量探讨自由、意识、文明存续——通过情节而非说教。

### 四、沙雕搞笑向

**反差笑点**：严肃规则 + 无厘头日常；魔王守劳动法、修仙用微积分。

**弱化冲突痛感**：降低牺牲与致郁；误会式危机优于生死虐点。

**反套路**：可无金手指、无救世；摆烂躺平也可推进情节。

### 五、奇幻东方向

**国风元素改造**：春联、门神、孟婆、轮回等叠加反常识规则。

**中式氛围感**：庙会、老宅、节气、道观、夜雨街巷。

**传统概念重构**：鬼未必害人、神仙可能打工、投胎看体验值。

### 六、言情情感向

**规则绑定情感矛盾**：遗忘、寿命交换、梦境相见、心动数值——制造拉扯。

**情感优先级**：世界观服务恋爱；对话与心理戏重于设定科普。

**甜度/虐度可控**：按大纲标注控制代价轻重与误会深度。

### 七、末世生存向

**生存规则闭环**：资源从哪来、危险如何触发、违反规则付什么代价。

**人性冲突**：极端环境下信任、背叛、牺牲须具体可感。

**危机分层**：本章处于物资危机 / 关系危机 / 真相危机哪一层，勿乱跳。

## 题材适配示例（同一设定不同写法）

「记忆可以交易」在本章：
- 惊悚：交易后主角记不清自己名字；黑市经纪人微笑
- 治愈：主角卖掉痛苦记忆，亲人终于笑了
- 沙雕：摊位叫卖「童年暑假记忆，买一送一」
- 科幻：云端记忆编号被批量格式化

## 输出约束

- 纯正文；无章节标题行、无作者按语
- 续写只输出新增段落；一次生成写完整章
- 紧接上文语气、人称与已揭示规则
