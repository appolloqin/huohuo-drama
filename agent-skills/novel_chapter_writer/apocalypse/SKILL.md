# 末世文 · 章节正文 Skill

撰写 / 续写**末世尊章节正文**。规则自洽、资源有数、人性非脸谱、节奏持续加压。

---

## 一、世界观与规则（正文执行）

### 灾变与规则
- 已确立的灾难逻辑**前后一致**：爆发→扩散→环境→限制
- 异变/怪物/异能：**触发、弱点、上限、代价、迭代**写清后不得随意改
- 规则**公平不双标**；杜绝无铺垫的战力崩坏

### 资源体系
- **严格卡储量**：食物、水、药、燃料、弹药、零件、情报、配额
- 用资源差驱动冲突：紧缺时的交换、背叛、假意合作
- 禁止凭空变物、无限背包

---

## 二、末世人设（正文执行）

### 幸存者特质
- 被末世改变的人性：谨慎、多疑、感恩迟钝、利益优先、情绪克制
- **非黑即灰**：极端下背叛、交换、牺牲、假意合作、恩将仇报、抱团冷漠
- 禁：纯白圣母、无脑善良、危机时强行送死式利他（除非人设已铺垫且付代价）

### 强者逻辑
强在：**警觉、底线取舍、风险评估、果断止损、不因情绪误事**
- 异能数值不是唯一强弱标准

### 团队群像
- 职能互补：战力、侦查、医疗、后勤、技术、指挥
- 写真实摩擦：分歧、猜忌、抱团取暖并存

可塑造类型：冷静求生 / 狠辣利己 / 团队守护 / 苟发育 / 智商布局

---

## 三、剧情结构（本章定位）

### 开局段（若本章属前期）
- 急开局、压迫感：混乱、伤亡、失联、资源断、人性露底

### 中期段（主体）
循环链：**搜集物资 → 遭遇危险 → 人际博弈 → 取舍牺牲 → 能力迭代 → 环境恶化**

### 后期段（若本章属后期）
格局：据点争夺、势力博弈、区域规则、真相揭露

---

## 四、冲突创作（核心）

### 三重冲突叠加（重要章节尽量两层以上）

1. **外部**：怪物、天灾、敌对势力、规则异变
2. **内部**：猜忌、资源不足、立场分歧、内鬼
3. **自我**：底线动摇、恐惧、异化、代价反噬

### 高压节奏
- **持续微量压迫**；少长篇无事闲聊
- 会写「暂时安全」→ 瞬间打破（声响、气味、库存、背叛）

### 人性冲突
- 避免悬浮道德审判；行为服务于生存
- 擅长：极端下的交换与取舍

---

## 五、异能 / 进化体系

### 代价（重中之重）
杜绝无代价金手指。代价类型：
- 身体负荷、精神透支、寿命损耗、情绪剥夺、异化反噬

**越强越痛**；成长有重量。

### 进化梯度
- 怪物、环境、异能**同步迭代**；环境倒逼进化
- 主角有软肋、有克制点；单挑/团战/逃生能力分开考量

## 六、细分题材正文侧重

| 细分 | 正文要点 |
|------|----------|
| 硬核求生 | 物资细节、取舍狼狈、现实惊险、少开挂 |
| 异能进化 | 体系阶梯、团战配合、势力博弈 |
| 规则诡异 | 未知恐惧、细思极恐、真相反转、留白 |
| 基地建设 | 据点升级、制度、资源循环、攻防布局 |
| 末世言情 | 绝境偏爱、生死相守、互相救赎；灾难服务情感 |

## 七、职业化避雷（每章自检）

- [ ] 无圣母崩盘（危机行为符合已建立人设与利益）
- [ ] 无资源无限
- [ ] 无战力通胀（升级有边界、怪物有制衡）
- [ ] 无秩序轻易回弹（末世社会逻辑持续冷硬）
- [ ] 无全员善恶脸谱

---

## 八、万能创作公式（章节内核）

```
秩序崩塌 + 稀缺资源枷锁 + 危险迭代 + 人性取舍 + 异能/生存代价 +（后期）真相反转
```

本章至少落实其中 **3 项**，且与上文不矛盾。

## 正文排版与语言

- 网文排版：段落自然变化，段间空一行；对话有口气
- 简体中文；禁 AI 套话
- 纯正文；续写只输出新增；紧接上文资源存量与人际信任状态
