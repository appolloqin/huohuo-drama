# 游戏文 · 章节正文 Skill

撰写 / 续写**游戏文章节正文**。系统自洽、数值有梯度、爽压平衡，规则博弈高于无脑碾压。

---

## 正文通用原则

1. **规则钉死**：等级、技能、道具、惩罚前后一致；冷却、负重、权限等硬性约束生效
2. **代价绑定**：复活、漏洞利用、通关皆有消耗、风险或伦理代价
3. **轻量化数值**：实战表现体现强弱，忌大段枯燥数字堆砌
4. **系统叙事**：面板、任务、广播推动剧情，减少说明书旁白
5. **网文排版**：段落长短自然变化，段间空一行；对话有口气；忌诗化碎行
6. **简体中文**；禁 AI 套话（值得注意的是、不禁、至关重要等）

## 二、游戏人设（正文执行）

### 主角

- **天赋**：万金油 / 输出 / 肉盾 / 辅助 / 潜行 / 解谜智力
- **身份**：普通玩家、内测大佬、重生、职业穿越、被迫入局
- **特质**：观察力强、拆解规则、计算收益风险、利用漏洞（付代价）

### 配角与对手

- **队友**：输出、坦度、辅助、侦查、后勤商人、解谜智囊互补
- **对手**：氪金土豪、阴险会长、独行猎杀、伪善通关、利己背叛
- **NPC**：商人、任务发布、隐藏导师、自主 AI、伪装怪物

### 内心冲突

虚实割裂、杀戮负担、牺牲他人挣扎、AI 情感抉择

## 三、剧情结构（正文执行）

### 开篇

网游：开服/重生、面板、新手任务、资源冲突

无限流：强制拉入、规则明示、新手副本压迫

穿单机：绑定角色、偏离剧本、隐藏分支

### 中期循环

接任务/进副本 → 探索机制 → 冲突 → 资源提升 → 隐藏线索 → 结算奖惩

升级：单人 → 组队 → 公会 → 跨区 → 幕后真相

### 结局趋向（章末可铺垫）

全服登顶 / 揭穿系统 / 逃离轮回 / 同化系统 / 打破规则 / 人机和解

## 四、细分赛道正文要点

### 传统/全息网游
升级打怪、装备掉落、公会团战、竞技场 PK、商业囤货；氪金与平民对抗逻辑合理

### 无限闯关死亡游戏
诡异/规则副本、心理博弈、人性抉择、高死亡压迫；双结局：牺牲通关 vs 全员保全

### 穿单机/穿书
原剧情对照、已知信息规避、改写命运；NPC 自主意识、剧情崩坏连锁

### 经营建造
资源开采、建筑升级、居民招募、防御、贸易；规划管理外交为主

### 电竞职业
团战配合、战术布局、英雄克制、赛场心理；战队磨合、联赛成长

## 九、避雷（正文硬约束）

- **数值崩坏自检**：战力梯度稳定，禁后期随便秒杀前期级敌人无铺垫
- **规则统一**：惩罚、冷却、道具效果不前后矛盾
- **杜绝无脑碾压**：强者有短板，通关伴随代价
- **赛道区分**：电竞不强行诡异；死亡游戏不纯无脑升级爽
- **控制设定篇幅**：随剧情解锁，开局不倾倒全部系统
